## ⚠️ 核心前提

> **Brand Registry 的门槛是商标，不是公司。** 没有商标就没有 Brand Registry，没有 Brand Registry 就没有 A+ Content、Brand Analytics、Vine 计划、Manage Your Experiments。商标是品牌运营的地基。

## Module 1：商标申请策略

### 1.1 目标市场 vs 注册局对照

| 销售市场 | 商标注册局 | 审核周期 | 参考费用（官方费） |
|---------|----------|---------|----------------|
| Amazon US | USPTO（美国专利商标局） | 8–12 个月（普通）/ 3–5 个月（IP Accelerator） | $250–$350/类 |
| Amazon UK / EU | EUIPO（欧盟知识产权局） | 4–6 个月 | €850 起（1类） |
| Amazon JP | JPO（日本特许厅） | 10–16 个月 | ¥12,000–¥28,000/类 |
| Amazon CA | CIPO（加拿大） | 18–24 个月 | CAD $458/类 |
| 中国工厂保护 | CNIPA（国家知识产权局） | 9–12 个月 | ¥300/类 |

> 建议优先注册 **US + EU** 覆盖主力市场，中国注册用于防工厂抢注。

### 1.2 商标类别选择（尼斯分类）

跨境电商最常用类别：

| 尼斯类别 | 覆盖内容 | 适用产品举例 |
|---------|---------|------------|
| **第 20 类** | 家具、家居装饰、收纳 | 收纳盒、椅子、架子 |
| **第 21 类** | 厨房用具、清洁工具 | 水杯、锅具、毛刷 |
| **第 25 类** | 服装、鞋、帽 | 服装类目 |
| **第 28 类** | 玩具、运动器材 | 健身器材、玩具 |
| **第 35 类** | 零售服务 | 建议同时注册保护销售行为 |

**原则**：注册你的产品所在类 + 第 35 类（零售服务），双重保护。

### 1.3 IP Accelerator 快速通道（Amazon 官方项目）

- Amazon 与认证律师事务所合作，申请期间即可使用 Brand Registry 功能
- 等待 USPTO 审核（通常 8–12 个月）期间不再是空档期
- 操作路径：`Brand Registry → IP Accelerator → 选择律所 → 律所代理申请`
- 适合：产品已在售、急需解锁品牌功能的卖家

## Module 2：Amazon Brand Registry 申请

### 2.1 申请条件

```
必须满足：
✅ 拥有活跃的商标注册号（或通过 IP Accelerator 申请中）
✅ 商标含文字（图形商标不够，需含品牌文字）
✅ 商标与品牌名在 Amazon 上一致
✅ 有效的 Seller Central 或 Vendor Central 账户
```

### 2.2 申请步骤

```
Step 1：进入 brandregistry.amazon.com
Step 2：点击 "Enroll a brand"
Step 3：填写信息：
  - 品牌名称（与商标完全一致）
  - 商标注册号
  - 注册局（USPTO / EUIPO / JPO 等）
  - 产品图片（含品牌 logo）
  - 产品类别
Step 4：Amazon 发送验证码至商标注册人邮箱
Step 5：输入验证码，提交审核
Step 6：审核通过（通常 3–10 个工作日）
```

### 2.3 注册后解锁的功能

| 功能 | 说明 | 价值 |
|------|------|------|
| **A+ Content** | 富媒体产品描述，提升转化 10–15% | 高 |
| **Premium A+ Content** | 视频模块、交互图，仅品牌卖家可用 | 高 |
| **Brand Analytics** | 搜索词报告、市场篮子分析、人群洞察 | 极高 |
| **Manage Your Experiments** | Listing A/B 测试 | 高 |
| **Vine 计划** | 邀请 Top Reviewer 写评论 | 中 |
| **Sponsored Brands 广告** | 品牌广告位（搜索结果顶部） | 高 |
| **Brand Stores** | 品牌旗舰店 | 中 |

## Module 3：防跟卖与侵权打击

### 3.1 跟卖类型识别

| 类型 | 特征 | 严重程度 |
|------|------|---------|
| **正常跟卖** | 卖真品，合法，但抢 Buy Box | 中 |
| **仿冒品** | 卖假货，用你的 ASIN，产品质量差 | 高 |
| **盗图跟卖** | 用你的主图但卖不同产品 | 高 |
| **品牌劫持** | 修改你的 Listing 内容 | 极高 |

### 3.2 反跟卖操作 SOP

**第一步：监控（每日）**

```
操作路径：Seller Central → Inventory → Manage All Inventory
→ 找到被跟卖的 ASIN → 查看 "Other Sellers on Amazon"
→ 记录：卖家 ID、价格、Seller Rating
```

**第二步：警告信（72小时内发送）**

```
警告信模板（英文）：

Subject: Cease and Desist – Unauthorized Sale of [Brand Name] Products

Dear Seller,

We are writing to notify you that you are selling counterfeit/unauthorized 
products using our registered trademark [BRAND NAME] (Registration No. XXXXXXX).

This listing (ASIN: BXXXXXXXXX) belongs exclusively to [Company Name]. 
Your unauthorized sale infringes our intellectual property rights under 
15 U.S.C. § 1125 and Amazon's Anti-Counterfeiting Policy.

We demand you immediately:
1. Remove all listings using our brand name or product images
2. Cease all sales of counterfeit/unauthorized [Brand Name] products

Failure to comply within 48 hours will result in a formal complaint 
to Amazon and legal action.

Sincerely,
[Your Name] | [Company Name] | [Contact]
```

**第三步：Amazon Report a Violation（品牌备案后可用）**

```
操作路径：
brandregistry.amazon.com → Protect → Report a Violation
→ 搜索 ASIN 或卖家 → 选择侵权类型：
  · Counterfeit（仿冒）
  · Trademark infringement（商标侵权）
  · Copyright infringement（版权侵权）
  · Patent infringement（专利侵权）
→ 上传证据（产品对比图、商标证书）
→ 提交
→ Amazon 通常 3–5 个工作日处理
```

### 3.3 Transparency 透明计划（防仿冒终极方案）

- 每件产品贴唯一防伪码（条形码），买家可扫码验真伪
- Amazon 在 FBA 入库时扫描，无码产品直接拦截
- 费用：$0.01–$0.05/件（按产品单价分级）
- 适合：已遭受严重仿冒、客单价 ≥$20 的品牌

**申请入口**：`brandregistry.amazon.com → Protect → Transparency`

## Module 4：品牌保护 ROI 分析

```
品牌备案投入产出估算

投入：
- 商标注册（USPTO）：$300–$500（一次性，10年有效）
- 律师费（可选）：$500–$2,000
- Transparency 计划：$0.01–$0.05/件

产出（品牌备案后）：
- A+ Content 提升转化 10–15%
- Sponsored Brands 广告位独占
- Brand Analytics 数据价值（替代 $100+/月的第三方工具）
- 跟卖损失减少（通常跟卖会侵蚀 15–30% 销售额）

结论：
- 商标费用通常在 3–6 个月内通过转化提升回收
- 若已有跟卖，品牌备案后首月即可看到效果
```
