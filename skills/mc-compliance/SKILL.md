## 触发条件

当用户提到以下任意情形时使用本技能：

- "这个产品能不能卖到美国/欧洲/日本" / "合规审查" / "compliance check"
- "需要什么认证" / "FDA/CE/FCC 要求"
- "Amazon 能上架吗" / "Listing 有没有违规"
- "跨境合规" / "出口合规" / "cross-border compliance"
- "知识产权风险" / "会不会侵权" / "IP risk"
- "FBA 标签要求" / "包装合规" / "Prop 65"
- "HS Code" / "关税税率" / "这个产品关税多少" / "Section 301"
- "VAT 注册" / "要不要注册 VAT" / "税务合规" / "Sales Tax"
- "转让定价" / "税务架构" / "离岸公司"
- 提供了产品信息并询问海外市场准入问题

## Module 1: 产品准入审查

### 核心原则

不同国家/地区对同一产品的准入要求差异极大。**认证缺失是跨境卖家最常见的封号原因。**

### 产品类别 × 市场准入矩阵

#### 美国 (US)

| 产品类别 | 主管机构 | 必要认证/注册 | 关键要求 |
|----------|---------|--------------|---------|
| 食品 | FDA | FDA Food Facility Registration + Prior Notice | 进口前需提交 Prior Notice；标签符合 21 CFR Part 101 |
| 化妆品 | FDA | MoCRA 注册（2024 年起强制） | 成分须在 FDA VCRP 登记；禁用成分清单见 21 CFR 700 |
| 膳食补充剂 | FDA | FDA Facility Registration | 标签必须含 "This statement has not been evaluated by the FDA. This product is not intended to diagnose, treat, cure, or prevent any disease." |
| 消费电子 | FCC | FCC ID / SDoC | 所有辐射电子设备需 FCC 认证；蓝牙/WiFi 设备必须 FCC ID |
| 儿童产品 | CPSC | CPC (Children's Product Certificate) | 12 岁以下儿童产品需第三方 CPSC-accepted lab 检测；铅含量 ≤ 100ppm |
| 一般消费品 | CPSC | GCC (General Certificate of Conformity) | 成人产品需 GCC；基于合理测试计划 |
| 激光产品 | FDA CDRH | Accession Number | 激光产品（含激光笔）需向 FDA CDRH 报告 |
| 杀虫/消毒产品 | EPA | EPA Registration | 声称 "kills germs/bacteria" 的产品必须 EPA 注册 |

**实例：** 一个带蓝牙功能的儿童智能手表，同时需要 FCC ID（蓝牙辐射）+ CPC（儿童产品）+ 铅含量检测 + 小零件窒息风险评估。漏掉任何一项都可能被海关扣货或 Amazon 下架。

#### 欧盟 (EU)

| 产品类别 | 核心法规 | 必要认证/标识 | 关键要求 |
|----------|---------|-------------|---------|
| 几乎所有消费品 | 相关 EU 指令 | CE Marking | CE 标识是进入欧盟市场的"护照"；需自行或通过 NB 评估符合性 |
| 电子电气产品 | LVD + EMC | CE (LVD 2014/35/EU + EMC 2014/30/EU) | 低压指令 + 电磁兼容；需技术文档 + DoC 声明 |
| 含化学物质产品 | REACH | REACH 合规（SVHC ≤ 0.1%） | 高关注物质清单每年更新；卖家是 "Only Representative" 或需指定一个 |
| 电子电气产品 | RoHS | RoHS 2011/65/EU | 铅、汞、镉等 10 种有害物质限制 |
| 电子电气产品 | WEEE | WEEE 注册 + 标识 | 每个 EU 成员国单独注册；产品须印"打叉垃圾桶"标识 |
| 医疗器械 | EU MDR 2017/745 | CE (通过 Notified Body) | 2024 年起全面执行 MDR；Class I 以上需 NB 参与 |
| 玩具 | Toy Safety Directive 2009/48/EC | CE + EN 71 检测 | 机械物理 + 化学 + 可燃性检测 |
| 化妆品 | EU Cosmetics Regulation 1223/2009 | CPNP 通报 + PIF | 上市前须在 CPNP 系统通报；需完整 Product Information File |
| 食品 | General Food Law + 标签法规 | — | 标签须含过敏原信息（14 种）；营养成分表强制 |

**实例：** 一个 USB 充电的 LED 台灯出口欧盟，需要：CE（LVD + EMC）+ RoHS 检测报告 + REACH SVHC 声明 + WEEE 注册（每个目标国家）+ DoC 符合性声明 + 技术文档存档 10 年。成本预估：检测费 ¥8,000-15,000 + 各国 WEEE 注册费。

#### 英国 (UK)

| 要点 | 说明 |
|------|------|
| UKCA 标识 | 脱欧后英国自有标识，取代 CE（但目前延期接受 CE 标识至 2025 年底，需关注最新政策） |
| UK Responsible Person | 必须在英国境内指定一名 Responsible Person，负责合规文档 |
| UK REACH | 独立于 EU REACH，需单独注册 |
| WEEE 注册 | 在 UK Environment Agency 单独注册 |

**关键提醒：** UK 政策变化频繁，每次审查须确认 UKCA/CE 接受截止日期的最新状态。

#### 日本 (JP)

| 产品类别 | 主管机构/法规 | 必要认证 | 关键要求 |
|----------|-------------|---------|---------|
| 电气产品（特定） | 电気用品安全法 | PSE (菱形) | 特定电气用品（电源线、锂电池等）需第三方检测 + 菱形 PSE |
| 电气产品（一般） | 电気用品安全法 | PSE (圆形) | 一般电气用品可自主检测 + 圆形 PSE |
| 消费生活用品 | 消費生活用製品安全法 | PSC | 特定产品（婴儿床、打火机等）需 PSC 标识 |
| 食品 | 食品衛生法 | 厚生劳动省检疫 | 进口食品需在港口接受检疫检查 |
| 化妆品 | 薬機法 | 化妆品制造销售业许可 | 需日本境内持牌业者作为"制造销售业者" |
| 医疗器械 | 薬機法 | PMDA 认证 | 根据分类需不同级别认证 |
| 无线设备 | 电波法 | 技適マーク (TELEC) | 所有无线设备必须获得技術基準適合証明 |

**实例：** 一个带 WiFi 功能的空气净化器出口日本，需要：PSE（电气安全）+ 技適マーク（WiFi 无线）+ 食品衛生法适用性确认（如果接触食品/饮用水）。没有技適マーク的无线设备在日本使用属于违法。

#### 澳大利亚 (AU)

| 产品类别 | 主管机构 | 必要认证 | 关键要求 |
|----------|---------|---------|---------|
| 电气产品 | EESS | RCM Mark (SAA) | 电气安全 + EMC 合规 |
| 消费品安全 | ACCC | 符合 Australian Consumer Law | 强制安全标准 + 强制信息标准 |
| 食品 | FSANZ | 食品标准合规 | Food Standards Code 标签要求 |
| 治疗商品 | TGA | ARTG 注册 | 治疗性商品（含部分保健品）需 TGA 注册 |

### 准入审查执行步骤

1. **确定产品 HS Code** — 海关编码决定关税税率和监管要求
2. **识别所有适用法规** — 一个产品可能同时受多个法规管辖（如电子+儿童+无线）
3. **列出必要认证清单** — 区分"强制"和"推荐"认证
4. **评估认证获取难度** — 时间周期（2周-6个月不等）、费用范围、所需样品数量
5. **检查禁止/限制项** — 某些成分/材质/功能在目标市场可能完全禁止

## Module 2: Amazon/平台合规

### Amazon

| 合规项 | 说明 | 风险等级 |
|--------|------|---------|
| 受限产品 | Amazon Restricted Products List 覆盖数十个品类；部分需申请解锁（如杀虫剂、医疗器械、食品补充剂） | 高 |
| 产品认证文件 | Amazon 可能随时要求提供合规文件（CPC、GCC、SDS、检测报告）；无法在 48h 内提供 → 下架 | 高 |
| Listing 政策 | 标题 ≤ 200 字符；禁止全大写；禁止促销信息（"Sale"、"Free Shipping"）出现在标题/图片 | 中 |
| A+ Content 规则 | 不能包含公司联系信息、外部链接、定价信息、"best seller" 等未验证声明 | 中 |
| Review 政策 | 禁止以任何方式激励好评（包括产品插卡暗示好评返现）；违反 = 永久封号 | 高 |
| Brand Registry | 需有目标国家注册商标；R 标或 TM 标（IP Accelerator 可加速） | 中 |
| Product Safety | 锂电池需 UN38.3 + MSDS；危险品需 SDS (Safety Data Sheet) | 高 |
| 变体政策 | 不同产品不能通过变体合并到同一 Listing（操纵排名 = 封号） | 中 |

**高频封号原因（按严重程度排序）：**

1. **刷单/虚假评价** — 永久封号，几乎无法申诉
2. **缺少合规文件** — 限期提供，逾期下架
3. **侵权（商标/专利/版权）** — ASIN 下架 + 可能冻结资金
4. **产品安全投诉** — 客户安全相关投诉触发紧急下架
5. **Review 操纵（插卡返现）** — 2023 年起 Amazon 大规模扫荡

### Shopee

| 市场 | 关键合规项 |
|------|-----------|
| 通用 | 禁售清单因市场而异；需逐一确认每个站点 |
| 东南亚各站 | 食品需当地 FDA 注册（如泰国 อย.）；化妆品需当地注册号 |
| Listing 规则 | 标题含垃圾关键词 → 降权；主图必须白底 |
| 价格规则 | 不能设置明显不合理的高原价再打折（虚假促销） |

### TikTok Shop

| 合规项 | 说明 | 风险等级 |
|--------|------|---------|
| 产品限制 | 禁售清单严格，包括二手商品、处方药、武器等 | 高 |
| 内容规则 | 短视频/直播中不能做未经证实的功效声明 | 高 |
| 健康类声明 | 保健品/化妆品声称治疗效果 → 直接下架 + 封店 | 高 |
| 前后对比 | "Before/After" 对比图在美妆/保健品类受严格限制 | 中 |
| 价格虚标 | 人为抬高原价再打折 → 违规 | 中 |

### Temu

| 合规项 | 说明 | 风险等级 |
|--------|------|---------|
| 质量标准 | Temu 对产品质量投诉率有严格阈值；超过 → 罚款 + 下架 | 高 |
| 定价规则 | 极低价竞争策略，但不能低于成本价恶意竞争 | 中 |
| 退货率 | 退货率过高的产品被强制下架 | 中 |
| 合规文件 | 入驻时需提供营业执照 + 相关产品认证 | 高 |

## Module 3: Listing 文案合规

### 高风险声明类型（全球通用）

| 声明类型 | 禁止/限制用法示例 | 合规替代方案 |
|----------|------------------|-------------|
| 医疗声明 | "治疗湿疹" "cure acne" "治愈失眠" | "帮助舒缓肌肤不适" "helps soothe skin" |
| 绝对化声明 | "100% effective" "最好的" "No.1" | "highly rated" "深受好评" |
| 安全承诺 | "完全无毒" "100% safe" | "符合 XX 安全标准" "tested to XX standard" |
| 环保声明 | "eco-friendly"（无认证） "全天然" | "made with recycled materials（具体比例）" |
| 减重声明 | "7 天瘦 10 斤" "lose weight fast" | "配合健康饮食和运动使用" |
| 防水等级 | "waterproof"（未经 IP 测试） | "water-resistant (IPX4 rated)" |

### 美国 FTC 规则细化

| FTC 规则 | 要求 | 违规后果 | 高风险短语 |
|----------|------|---------|-----------|
| Substantiation | 所有客观声明必须有"合理依据" | 罚款 + 禁令 | "clinically proven" "scientifically tested"（无检测报告时） |
| Endorsements | 付费评价/推荐必须披露关系 | 罚款最高 $50,120/次 | "as recommended by doctors"（无真实背书时） |
| Made in USA | 产品"全部或几乎全部"在美制造才能声称 | FTC 调查 + 罚款 | "Made in USA"（中国组装产品绝对不能用） |
| Comparative Ads | 对比必须公平、准确、可验证 | 竞争对手诉讼 | "better than [Brand X]"（无检测数据时） |
| Environmental | 环保声明需具体且有依据 (FTC Green Guides) | 罚款 + 强制更正 | "biodegradable" "carbon neutral"（无第三方认证） |
| Children's | COPPA: 不能收集 13 岁以下儿童数据 | 巨额罚款 | 面向儿童的产品页收集邮箱/生日 |

**中国卖家常踩的 FTC 坑：**
- "Best seller" — 除非你真的是 Amazon BSR #1 并有截图证据
- "Award-winning" — 必须有真实获奖记录
- "Doctor recommended" — 需要有真实医生背书文件
- 产品图片展示效果与实际不符（如过度 PS）

### 欧盟消费者保护指令

| 指令 | 要求 | 高风险场景 |
|------|------|-----------|
| Unfair Commercial Practices Directive 2005/29/EC | 禁止误导性和攻击性商业行为 | 虚假倒计时、虚构库存紧张、虚假好评 |
| Consumer Rights Directive 2011/83/EU | 14 天无理由退货权（远程销售） | Listing 中不能暗示"不退不换" |
| Price Indication Directive | 促销价须标注此前 30 天最低价 | "50% OFF"但原价是虚构的 |
| EU Omnibus Directive (2022) | 在线评价必须说明是否经过验证 | 精选好评展示而隐藏差评 |

### 日本 景品表示法

| 违规类型 | 说明 | 高风险短语（日文 Listing） |
|----------|------|--------------------------|
| 優良誤認表示 | 夸大产品品质/性能 | "業界初" "特許技術"（无证据时） |
| 有利誤認表示 | 虚假的价格/交易条件 | "通常価格 ¥10,000 → 今だけ ¥3,000"（通常价格不存在时） |
| 打消表示不足 | 附加条件的字体太小或不显眼 | 促销条件用极小字号标注 |

**日本市场特别注意：** 日本消费者厅对景品表示法执法非常严格。2023-2024 年多起中国跨境卖家因虚假折扣被处罚的案例。

### 各市场图片要求

| 市场/平台 | 主图要求 | 禁止内容 |
|-----------|---------|---------|
| Amazon（全站） | 纯白背景 (RGB 255,255,255)；产品占比 ≥ 85%；无文字/Logo/水印 | 与产品无关的配件、生活场景（主图） |
| Amazon US | 人体模特图不能是绘图/插画 | "Amazon's Choice" 等平台标识 |
| Amazon JP | 同上 + 日文标签需清晰可读 | 过度修图导致与实物差异大 |
| TikTok Shop | 清晰产品图；可含少量文字 | 前后对比图（美妆/保健类） |

## Module 4: 知识产权风险

### 中国卖家高频 IP 陷阱

| 陷阱类型 | 说明 | 真实案例 |
|----------|------|---------|
| 通用词被注册 | 国内认为的通用词在海外被注册为商标 | "Selfie Stick" 在美国曾被注册商标；"Fidget Spinner" 的设计专利纠纷 |
| 外观设计侵权 | 产品外观与已有 Design Patent 过于相似 | 模仿 Dyson 吹风机外观 → 被起诉 |
| 实用专利侵权 | 产品功能/结构落入他人专利保护范围 | 手机支架的折叠机构专利 |
| 商标抢注 | 自己品牌名在目标国家被他人抢注 | 国内品牌进军美国发现商标已被注册 |
| 版权侵权 | 使用他人图片/插画/卡通形象 | Listing 图片含迪士尼角色 → DMCA 下架 |
| 包装设计侵权 | 包装与知名品牌 Trade Dress 过于相似 | 模仿苹果包装极简风格 + 类似字体 |

### IP 排查步骤

1. **商标搜索**
   - 美国: USPTO TESS (tmsearch.uspto.gov)
   - 欧盟: EUIPO eSearch plus
   - 英国: UK IPO Trade Marks Journal
   - 日本: J-PlatPat
   - 全球: WIPO Global Brand Database
   - **搜索范围：** 产品名称、品牌名、关键卖点词、Listing 标题中的核心词

2. **专利搜索**
   - Google Patents (patents.google.com) — 覆盖全球主要专利局
   - 搜索范围：产品核心功能、外观设计、关键结构
   - **重点关注 Design Patent** — 外观设计专利是中国卖家被诉最多的类型

3. **Amazon 侵权投诉记录**
   - 在 Amazon 搜索同类产品，查看是否有大量卖家下架（可能存在 Patent Troll 活跃）
   - 关注 Amazon Brand Registry 中的品牌保护投诉

4. **规避建议**
   - 品牌名在目标市场注册商标（USPTO 注册周期 12-18 个月，费用 $250-350/类）
   - 考虑 Amazon IP Accelerator 加速 Brand Registry 审批
   - 产品外观做差异化设计，避免"致敬"知名品牌
   - Listing 文案不使用他人商标词作为关键词（即使是竞品名）

### 收到侵权投诉的应对

| 投诉类型 | 紧急程度 | 应对步骤 |
|----------|---------|---------|
| 商标侵权 (Trademark) | 高 | 1. 立即下架涉诉 ASIN 2. 联系投诉方协商 3. 修改品牌/关键词后重新上架 |
| 专利侵权 (Patent) | 高 | 1. 下架 2. 找专利律师评估 3. 决定是否申诉/和解/规避设计 |
| 版权侵权 (Copyright) | 中 | 1. 替换侵权素材 2. 提交申诉 3. 确保所有素材来源合法 |
| Trade Dress | 中 | 1. 修改包装/外观设计 2. 强调差异化 |

## Module 5: 物流与标签合规

### FBA 标签要求

| 标签类型 | 要求 | 不合规后果 |
|----------|------|-----------|
| FNSKU 标签 | 每个产品单位必须贴 FNSKU 条码；覆盖原有条码 | 入库拒收或与他人库存混合（Commingled Inventory 风险） |
| 窒息警告 | 所有含塑料袋的包装（厚度 < 1 mil）必须印/贴窒息警告 | FBA 入库拒收；可能触发安全投诉 |
| Country of Origin | "Made in China" 必须清晰标注在产品或包装上 | 美国海关扣货（19 USC §1304） |
| 重量/尺寸 | 超过 50 lbs 需贴 "Team Lift" 标签 | FBA 收取额外处理费 |

**窒息警告标准文本（美国）：**
```
WARNING: To avoid danger of suffocation, keep this plastic bag
away from babies and children. Do not use this bag in cribs,
beds, carriages, or playpens. This bag is not a toy.
```

### 产品标签各市场要求

| 市场 | 语言要求 | 必标信息 |
|------|---------|---------|
| 美国 | 英文 | 产品名、制造商/进口商名称地址、原产国、净含量（重量/体积）、安全警告 |
| 欧盟 | 销售国官方语言（可多语言） | 产品名、制造商+EU Responsible Person 名称地址、原产国、CE 标识、WEEE 标识 |
| 英国 | 英文 | 同欧盟 + UK Responsible Person 地址（不能用 EU 地址替代） |
| 日本 | 日文 | 产品名、进口商名称地址、原产国、PSE/技適マーク（如适用）、使用注意事项 |
| 澳大利亚 | 英文 | 产品名、供应商名称地址、原产国、RCM Mark（如适用） |

**常见标签错误：**
- EU 标签只有英文没有当地语言 → 被拒入境
- 日本标签用简体中文代替日文 → 不被接受
- "Made in PRC" 代替 "Made in China" → 美国海关可能质疑

### 包装法规

| 法规 | 适用市场 | 要求 |
|------|---------|------|
| EU Packaging Directive 94/62/EC | 欧盟 | 包装材料回收标识；各成员国 EPR (Extended Producer Responsibility) 注册 |
| 德国 VerpackG | 德国 | 必须在 LUCID 系统注册包装许可证 |
| 法国 AGEC Law | 法国 | Triman 标识 + 分类回收信息 |
| California Prop 65 | 美国加州 | 产品含加州已知致癌/生殖毒性物质需贴 Prop 65 警告 |
| TPCH (Toxics in Packaging) | 美国 19 州 | 包装材料中铅+汞+镉+六价铬总量 ≤ 100ppm |

**Prop 65 特别说明：** 这是中国卖家最常收到的合规投诉之一。Prop 65 涵盖约 900+ 种化学物质。很多常见材料（如 PVC、某些金属合金、含铅焊料）都可能触发。建议：
- 高风险品类（电子、珠宝、厨房用品）提前做 Prop 65 检测
- 如检测结果接近阈值，主动贴 Prop 65 警告比被起诉划算
- Prop 65 律师函 (Bounty Hunter Lawsuits) 和解费通常 $5,000-$50,000

### 电池/危险品运输

| 物品 | 运输要求 | 文件要求 |
|------|---------|---------|
| 锂电池（内置） | UN3481；需通过 UN38.3 测试 | UN38.3 测试报告 + MSDS/SDS |
| 锂电池（独立） | UN3480；限制更严格 | UN38.3 + SDS + 危险品申报 |
| 含液体产品 | 需防漏包装 | SDS（如适用） |
| 气雾剂 | 危险品分类；部分物流渠道拒收 | SDS + 危险品运输申报 |
| 磁性产品 | 航空运输需消磁或屏蔽 | 磁性检测报告 |

**FBA 危险品审查 (Hazmat Review)：** 含电池、液体、粉末、喷雾的产品在 FBA 入库前可能触发 Hazmat Review，需提供 SDS。审查周期 1-4 周，期间无法入库销售。建议提前准备 SDS 文件。

### 海关文件要求

| 文件 | 用途 | 关键信息 |
|------|------|---------|
| Commercial Invoice | 清关必备 | 发货人/收货人、HS Code、产品描述、数量、单价、总价、原产国、Incoterms |
| Packing List | 辅助清关 | 箱数、每箱内容、毛重/净重、尺寸 |
| HS Code | 确定关税税率 | 错误 HS Code → 关税计算错误 → 海关罚款 |
| Certificate of Origin | 部分产品必需 | 证明原产地；享受优惠关税（如有贸易协定） |
| 合规证书 | 海关抽检 | CE/FCC/CPC 等证书副本随货备查 |

### 门禁判定

| 综合结论 | 条件 |
|----------|------|
| ✅ 可上架 | 所有模块 🟢 或 🟡，无任何 🔴 项 |
| ⚠️ 需补充材料 | 存在 🟡 项但无 🔴 项；或 🔴 项可在短期内解决 |
| 🚫 高风险暂停 | 存在 🔴 项且短期无法解决（如缺少强制认证、明确侵权） |

### 优先处理排序

🔴 高风险项按以下优先级排序：
1. **产品安全/强制认证缺失** — 海关扣货 + 平台下架 + 法律风险
2. **知识产权侵权** — 账号冻结 + 资金冻结 + 法律诉讼
3. **Listing 违法声明** — 平台下架 + 监管罚款
4. **标签/包装不合规** — 海关退运 + 入库拒收

## Module 7: 税务合规与 HS Code 管理

> 2026 年各国税务执法趋严，跨境卖家面临 VAT/GST 注册义务、所得税合规、HS Code 错误导致的关税风险。本模块聚焦"不被罚"和"不多交"。

### 7.1 HS Code 识别与归类

HS Code（Harmonized System Code）决定关税税率，归类错误可能导致多交税或被海关处罚。

**HS Code 查询流程：**

```
Step 1: 确定产品核心属性
  → 材质（金属/塑料/纺织品/电子...）
  → 功能（家用/工业/运动/装饰...）
  → 组成（单一材质 vs 复合材质）

Step 2: 查找 HS Code
  → 数据探测：搜索 "{产品英文名} HS code" / "harmonized tariff schedule {product}"
  → 参考来源：USITC HTS Search / EU TARIC / 中国海关编码查询
  → 同类产品参考：查看同品类卖家使用的 HS Code

Step 3: 验证与交叉检查
  → 前 6 位国际统一（HS），后 2-4 位各国不同（HTS/CN Code）
  → 检查该 HS Code 下的关税税率是否合理
  → 注意：同一产品在不同国家可能归入不同 HS Code
```

**常见归类陷阱：**

| 产品类型 | 常见错误 | 正确归类建议 |
|---------|---------|------------|
| 带电池的产品 | 按产品主体归类 | 需考虑是否按"电池产品"单独归类 |
| 套装/组合产品 | 随意选一个组件的 HS Code | 按"赋予整体基本特征"的组件归类 |
| 多功能产品 | 按次要功能归类（税率低） | 必须按主要功能归类 |
| 纺织品 | 忽略材质比例 | 按重量占比最高的纤维归类 |
| 食品/保健品 | 按"普通食品"归类 | 含特殊成分可能归入"膳食补充剂"不同税率 |

### 7.2 各国关税与 Section 301 清单

**美国（US）关税框架：**

```
总关税 = MFN 基础税率 + Section 301 附加税 + AD/CVD（如适用）

MFN 税率：0-20%（取决于 HS Code）
Section 301（中国产品附加关税，2026 年现行清单）：
  List 1: 25%（~818 个税号）
  List 2: 25%（~279 个税号）
  List 3: 25%（~5,745 个税号）
  List 4a: 7.5%（~2,500 个税号，部分被排除）

  ⚠️ 2025 年新增：
    电动汽车：100%
    半导体：50%
    电池/关键矿产：25%
    钢铝：25%
    太阳能电池：50%
    港口起重机：25%
    个人防护设备：25%
    注射器/针头：50%

确认方法：
  → 搜索 "USTR Section 301 exclusions {HS code}"
  → 检查是否有临时排除（exclusion）
  → 注意排除的到期日
```

**欧盟（EU）关税 + VAT：**

```
进口环节：
  关税 = CIF 价值 × 关税税率（0-12%，取决于 CN Code）
  进口 VAT = (CIF + 关税) × VAT 税率

各国 VAT 标准税率：
  德国: 19%  |  法国: 20%  |  意大利: 22%
  西班牙: 21%  |  荷兰: 21%  |  波兰: 23%
  英国: 20%（已脱欧，单独申报）

注意：部分品类有降低税率（如童装、食品、书籍）
```

**日本关税 + 消费税：**

```
关税 = CIF × 关税税率（0-15%，纺织品可达 10-15%）
消费税 = (CIF + 关税) × 10%

特殊：
  · 个人进口（≤ ¥16,666）免关税免消费税
  · 商业进口无免税额
  · EPA 协定国家（东南亚等）可享受优惠税率
```

### 7.3 VAT/GST 注册义务

**各市场注册阈值与义务：**

| 市场 | 注册阈值 | 触发条件 | 申报频率 |
|------|---------|---------|---------|
| 英国 | £0（非英国卖家必须注册） | 在英国存储库存或向英国消费者销售 | 季度 |
| 德国 | €0（非德国卖家必须注册） | 在德国存储库存（含 FBA） | 月度 |
| 法国 | €0（非法国卖家必须注册） | 在法国存储库存 | 月度 |
| 美国各州 | 各州不同（$100K-$500K） | Economic Nexus：达到销售额/交易数阈值 | 季度/月度 |
| 加拿大 | CAD 30,000/年 | 全球收入或加拿大销售超阈值 | 季度 |
| 澳大利亚 | AUD 75,000/年 | 全球营业额或澳洲销售超阈值 | 季度 |
| 日本 | ¥10,000,000/年 | 上上一课税年度销售超阈值 | 年度 |

**Amazon FBA 卖家特别注意：**

```
FBA 库存存储 = 在该国有"固定场所" = 触发 VAT 注册义务

常见场景：
  · 使用 Pan-EU FBA → 需在所有存储国注册 VAT（可能 5-7 个国家）
  · 使用 EFN（European Fulfillment Network）→ 仅需注册本国 + 销售超阈值国
  · 使用 CEE（中东欧扩展）→ 额外触发波兰/捷克 VAT

建议：
  → 新卖家先用 EFN 减少 VAT 注册负担
  → 销量上来后再切 Pan-EU 享受更快配送
```

### 7.4 转让定价与税务架构

> 适用于年销售额 > $500K 或多国运营的卖家。

**基本概念：**

```
中国公司（采购）→ 离岸公司（香港/新加坡）→ 目标市场销售

需要解决：
1. 各环节的利润分配是否合理？（转让定价）
2. 哪个实体承担什么功能和风险？（功能分析）
3. 是否有实质性运营？（经济实质法）
```

**税务架构建议（按规模）：**

| 年销售额 | 建议架构 | 核心考虑 |
|---------|---------|---------|
| < $200K | 个人/中国公司直接卖 | 简单，合规成本低 |
| $200K-$1M | 中国公司 + 目标市场 VAT 注册 | 满足 VAT 义务即可 |
| $1M-$5M | 中国公司 + 香港公司 + 目标市场实体 | 合理利润分配，降低整体税负 |
| > $5M | 专业税务顾问设计多层架构 | 转让定价文档、经济实质、CbCR |

**⚠️ 红线提醒：**
- 中国 CRS 信息交换：境外账户信息自动回传
- 香港经济实质法：空壳公司无法享受税收优惠
- 美国 FATCA：有美国银行账户必须申报
- 本技能仅提供框架性建议，**具体税务架构必须咨询专业税务师**

### 7.5 平台代扣代缴

各平台代扣代缴范围不同，卖家需了解哪些税已被平台处理：

| 平台 | 代扣代缴范围 | 卖家仍需自行处理 |
|------|------------|----------------|
| Amazon US | 美国 Sales Tax（所有州 Marketplace Facilitator） | 所得税、进口关税 |
| Amazon EU | 英国 VAT（≤£135）、EU OSS 国（≤€150）| 大额订单 VAT、进口 VAT、所得税 |
| Shopee | 部分市场 GST/VAT | 因市场而异，需确认当地规则 |
| TikTok Shop | 美国 Sales Tax | 所得税、进口关税 |
| Temu | 平台全托管模式下大部分税务由平台处理 | 中国公司所得税 |
