## 触发条件

当用户提到以下任意情形时使用本技能：

- "目标市场洞察" / "海外消费者画像" / "老外怎么买东西"
- "文化禁忌" / "本地化" / "这个产品能卖到美国/日本/欧洲吗"
- "跨境选品要注意什么文化差异" / "这个包装/颜色/命名在当地合适吗"
- "Amazon/Shopee/TikTok Shop 买家期望是什么"
- "Black Friday / 双十一 / Harbolnas 什么时候备货"
- "halal" / "清真" / "宗教要求" / "尺码转换" / "退货政策"
- "这个 listing 翻译有没有问题" / "直译还是本地化"

## 输入要求

启动本技能需要两个核心信息，缺一不可：

| 必填项 | 说明 | 示例 |
|--------|------|------|
| **品类** | 具体到二级品类 | 家用投影仪、女士瑜伽裤、宠物智能喂食器 |
| **目标市场** | 国家或区域 | US / JP / DE / 东南亚 / 中东 / LATAM |

如用户未提供，直接询问，不做假设。

可选补充信息：
- 价格带定位（低/中/高端）
- 目标平台（Amazon / Shopee / TikTok Shop / 独立站）
- 当前痛点（退货率高 / 差评集中 / 转化率低）

## 数据收集协议

### 工具探测

执行前检查当前环境中可用的搜索或浏览工具（WebSearch、WebFetch、agent-reach 等）。有搜索工具时优先使用；不限定具体工具名称，使用任何能访问外部实时数据的工具即可。

### 本技能数据清单

收集以下 4 类数据，每类执行 1-3 条搜索。根据目标市场使用对应语言关键词：

| 数据类型 | 搜索模板示例 |
|---------|------------|
| 目标市场消费趋势 | `[category] consumer trends [market] 2025`、`[品类] buying behavior [国家]` |
| 平台买家评价分析 | `[category] top reviews Amazon [market]`、`[品类] common complaints [平台]` |
| 文化禁忌与合规 | `[market] cultural taboos marketing`、`[国家] product packaging requirements` |
| 节日与营销日历 | `[market] shopping events calendar 2025`、`[国家] peak season e-commerce` |

### 降级处理

如当前环境无可用工具，跳过数据收集步骤，继续正常执行。在交付卡中标注：⚠️ 本次未检索实时数据，建议人工核验关键数据点。

### 引用格式

数据融入正文时用方括号数字标注来源序号：

> 美国消费者 73% 在下单前会阅读至少 3 条评论[1]

交付卡末尾附来源清单：

📊 数据来源
[1] 来源名称 / URL，检索日期 YYYY-MM-DD
[2] ...

如数据来自推断或无确凿来源，标注为：[推断，基于可比品类/历史趋势]，勿省略标注。

## 模块 1：目标市场消费者画像

不同市场的消费者是完全不同的物种。用中国市场的经验直接套到海外，是跨境卖家最常见的致命错误。

### 消费者画像维度

**人口统计（Demographics）**
年龄分布、收入水平、家庭结构、城乡分布——决定了消费力和品类偏好。

**消费心理（Psychographics）**
价值观排序、生活方式、品牌态度——决定了什么信息能打动他们。

**购物行为（Shopping Behavior）**
在哪里搜索、怎么比较、决策周期多长、退货习惯——决定了你的销售漏斗设计。

```markdown
## 消费者画像：{市场名} × {品类名}

### 核心人群
| 维度 | 描述 |
|------|------|
| 主力年龄段 | {年龄范围} |
| 收入水平 | {与品类定价的匹配度} |
| 家庭结构 | {独居/小家庭/大家庭，影响购买决策链} |
| 购物渠道偏好 | {线上 vs 线下，哪个平台为主} |

### 购物行为模式
| 行为 | 该市场特征 | 对中国卖家的影响 |
|------|-----------|-----------------|
| 搜索习惯 | {Google / 平台内搜索 / 社交媒体种草} | {SEO/广告策略启示} |

| 决策周期 | {冲动购买 vs 深度研究} | {listing 信息密度建议} |
| 评论依赖度 | {评分权重 / 图文评论 / 视频评论} | {review 运营优先级} |
| 退货习惯 | {退货率高低 / 退货原因 / 退货对信任的影响} | {退货政策设计} |

### 决策因素排序（该品类）
1. {最重要的购买决策因素}
2. {第二重要}
3. {第三重要}
...

### 价格感知
- 该品类当地主流价格带：{价格范围}
- "便宜"意味着：{消费者的负面联想，如质量差 / 不安全}
- "物有所值"的信号：{什么让他们觉得这个价格合理}
```

## 模块 2：文化规则与禁忌

每个市场都有"不成文的规矩"。违反了不会有人告诉你为什么卖不动，只会默默不下单。

### 主要市场文化规则速查

| 市场 | 文化底色 | 消费价值观 | 沟通风格 | 核心禁忌 |
|------|---------|-----------|---------|---------|
| **US** | 个人主义、多元化、实用主义 | 便利性 > 价格；重视个性化选择；sustainability 加分 | 直接、口语化、真实感（anti-corporate tone） | 种族/性别刻板印象、文化挪用、虚假环保宣称（greenwashing）、过度收集隐私 |
| **UK** | 低调讽刺、阶层意识、务实 | 品质导向但讨厌被"upsell"；幽默感是信任货币 | 含蓄、understated、self-deprecating | 过度夸张宣传、忽视 GDPR、粗暴的 push marketing |
| **DE** | 严谨、环保意识、质量至上 | Stiftung Warentest 测评影响极大；功能 > 花哨 | 精确、数据驱动、不喜欢空洞承诺 | 虚标参数、无环保认证、缺乏德语客服 |
| **FR** | 审美自信、生活品质、品牌故事 | 设计感和品牌叙事权重高；法国消费者对"美"有自己的标准 | 优雅但不矫揉，尊重法语 | 英文 listing 无法语版本、忽视法国节假日物流停摆 |
| **JP** | 细节美学、匠人精神、含蓄表达 | 品质 > 价格；包装 = 产品体验的一部分；季节感极强 | 敬语体系、间接表达、信息层次分明 | 过度直白推销、廉价感包装、忽视 JIS 标准、敷衍客服 |
| **KR** | 潮流导向、外貌经济、高标准 | 追逐新品、KOL/KOC 影响力极大、"性价比"概念不同（要质感） | 视觉优先、热情但要求完美 | 抄袭指控（韩国消费者极其敏感）、忽视韩文本地化 |
| **SEA 整体** | 家庭文化、宗教多元、移动优先 | 价格敏感但不等于只要便宜——要"划算" | 热情、社交化、喜欢互动式购物 | 宗教符号误用、忽视穆斯林市场斋月节奏、不提供 COD |
| **ID** | 穆斯林主体、社群导向、价格敏感 | Halal 认证几乎是必须项；COD 占比高 | 亲和力强、宗教敬语 | 猪相关元素、酒精成分未标注、无 BPOM 认证 |
| **TH** | 佛教文化、皇室尊崇、乐观包容 | 品牌意识正在觉醒，社交电商渗透率极高 | 可爱风（cute culture）、轻松 | 冒犯皇室（严格法律）、佛教符号商业化 |
| **VN** | 民族自豪感、年轻化、实用主义 | Facebook/TikTok 种草链条成熟；直播带货接受度高 | 直接、接地气 | 涉及中越政治敏感话题、领土争议相关 |
| **中东（GCC）** | 伊斯兰文化、家族荣耀、奢侈消费 | 品牌即身份；愿为品质和品牌溢价付费；礼品文化浓厚 | 尊重、正式、关系导向 | 非 halal 成分、女性暴露形象、猪/酒精/赌博元素、斋月期间的营销节奏 |
| **SA** | 保守但快速现代化（Vision 2030）、年轻人口 | 电商增长极快；对新品牌开放但要求本地化诚意 | 阿拉伯语优先、视觉丰富 | SASO 认证缺失、忽视阿拉伯语从右到左排版 |
| **UAE** | 国际化枢纽、多民族混居、高消费力 | 奢侈品消费全球领先；本地人 vs 外籍人口消费分层 | 英语+阿拉伯语双语 | 忽视 Emirates 标准认证、不支持 COD |
| **BR** | 热情、社交、分期文化 | Parcelamento（分期）是标配；Boleto 支付不可忽视 | 热情直接、社交媒体高度活跃 | 不支持分期付款、物流时效差无追踪、忽视巴西葡语（非欧洲葡语） |
| **MX** | 家庭核心、节日众多、品牌忠诚度高 | 熟人推荐权重大；对新品牌进入有"试探期" | 温暖、尊重、关系先行 | 忽视亡灵节等本地节日、用西班牙西语而非墨西哥西语 |

## 模块 3：本地化避坑指南

中国卖家出海最常犯的错误不是产品不好，而是"用中国逻辑卖给外国人"。

### 翻译 vs 本地化（直译 vs Localization）

| 层次 | 直译（Translation） | 本地化（Localization） |
|------|-------------------|---------------------|
| 语言 | 中文逐词翻成英文 | 用目标市场消费者的说话方式重新表达 |
| 卖点 | "高性价比" → "high cost performance" | "Incredible value without compromise" |
| 场景 | 中国家庭使用场景图 | 目标市场家庭/生活方式场景图 |
| 度量 | 厘米、克 | 英寸、磅（US）/ 厘米（EU）/ 具体市场适配 |
| 电压 | 220V | 110V（US/JP）/ 230V（EU/UK）/ 标注多电压支持 |

### 高频翻车案例

| 错误类型 | 典型表现 | 正确做法 |
|---------|---------|---------|
| **Chinglish 产品名** | "Intelligent Electric Warming Cup" | 研究 Amazon BSR 同品类 top listing 的命名方式 |
| **机翻 bullet points** | 语法错误、表达生硬、无法传递卖点 | 请母语 copywriter 或至少用 native speaker review |
| **中国模特/场景** | 产品图用中国面孔和中国家居场景 | 使用目标市场本地模特和生活场景 |
| **尺码不转换** | 只标注中国尺码 S/M/L | 提供该市场尺码对照表（US/UK/EU/JP 尺码体系各不相同） |
| **颜色命名** | "肉色" / "裸色" | 用目标市场的颜色命名方式（"nude" 在不同肤色文化中有争议，改用 "natural beige"） |
| **包装信息** | 全中文包装 | 目标市场语言包装，符合当地标签法规 |
| **节日错配** | 把中秋节促销推给美国消费者 | 跟随目标市场节日日历 |
| **客服时区** | 只有中国工作时间在线 | 覆盖目标市场工作时间，至少提供邮件 24h 内回复 |

### 图片/颜色/包装文化陷阱

```markdown
## 视觉本地化检查清单

### 颜色
| 颜色 | US | EU | JP | SEA | 中东 |
|------|----|----|----|----|------|
| 红色 | 警告/紧急/节日 | 警告/热情 | 吉祥/活力 | 吉祥但宗教场景慎用 | 无特殊禁忌 |
| 白色 | 纯洁/干净 | 纯洁/极简 | 丧葬（场景敏感） | 丧葬（部分国家） | 纯洁 |
| 绿色 | 环保/健康 | 环保 | 新鲜/自然 | 伊斯兰教圣色 | 伊斯兰教圣色（慎用于非宗教场景） |
| 黑色 | 高端/时尚 | 高端/正式 | 高端/神秘 | 丧葬（部分文化） | 高端 |
| 黄色 | 快乐/警告 | 快乐 | 皇室/勇气 | 皇室（TH/MY） | 无特殊禁忌 |

### 包装
- US：功能信息优先，环保材质加分，过度包装减分
- JP：包装是体验的一部分，精致度要求极高，开箱体验重要
- EU：环保包装几乎是强制要求（FR/DE 尤其严格）
- 中东：礼品属性强，包装精美度直接影响购买决策

### 人物形象
- US/EU：必须体现多元化（种族、体型、年龄）
- JP：偏好精致、干净、年轻化形象
- 中东：女性形象需保守（避免暴露，部分市场需戴头巾）
- SEA：各国审美差异大，不能用一套素材覆盖
```

### 客服期望差异

| 市场 | 响应时效期望 | 沟通风格 | 关键差异 |
|------|------------|---------|---------|
| US | 24h 内回复，live chat 加分 | 友好、解决导向、"customer is always right" | 退货宽松是常识，质疑退货会失去客户 |
| JP | 24h 内回复 | 极其礼貌、道歉先行、问题后解决 | 一次失误可能永远失去这个客户 |
| DE | 48h 可接受 | 精确、专业、不要套近乎 | 德语客服几乎是必须项 |
| SEA | 即时回复（live chat/WhatsApp） | 热情、耐心、反复确认 | 没有即时沟通渠道 = 不信任 |
| 中东 | 24h 内，WhatsApp 优先 | 尊重、正式、关系导向 | 斋月期间客服节奏需调整 |

### 退货政策文化差异

| 市场 | 退货文化 | 中国卖家常见失误 |
|------|---------|-----------------|
| US | 30天无理由退货是基线，Amazon 强制执行 | 退货政策写得像防贼——立刻掉转化率 |
| EU | 14天冷静期（法律要求） | 不了解当地消费者权益法，退货流程复杂 |
| JP | 退货率低但期望极高——不退货不代表满意，只是忍耐 | 误以为低退货率 = 高满意度，忽视差评积累 |
| SEA | 退货率较高，COD 拒收常见 | 不支持 COD 退货、物流追踪缺失 |

## 模块 4：平台消费者期望

同一个消费者在不同平台上的期望完全不同。Amazon 上要专业感，TikTok Shop 上要有趣。

### Amazon（US/EU/JP）

| 维度 | 买家期望 | 中国卖家常见差距 |
|------|---------|-----------------|
| 物流 | Prime 2天达是标配，FBA 几乎是必须项 | 自发货 7-15 天，买家直接差评 |
| Listing | 标题精准含关键词、5 点 bullet 信息密度高、A+ 页面专业 | 标题堆砌关键词、bullet 机翻、无 A+ 页面 |
| 评论 | 4.0+ 星是购买门槛、真实评论 > 数量 | 刷评被抓封号、缺乏合规 review 策略 |
| 退货 | 一键退货、全额退款、无需解释 | 退货流程复杂、客服推诿 |
| 品牌感 | Brand Store、品牌故事、品牌一致性 | 无品牌注册、店铺像杂货铺 |
| 价格 | 价格透明、频繁比价、coupon/deal 敏感 | 定价不做竞品分析、忽视 coupon 工具 |

### Shopee / Lazada（SEA）

| 维度 | 买家期望 | 中国卖家常见差距 |
|------|---------|-----------------|
| 价格 | 最低价 + voucher + free shipping 三件套 | 只拼价格不做平台活动，错过流量 |
| 互动 | 即时 chat 回复、问问题才下单 | 客服响应慢、不在线时段长 |
| 物流 | 本地仓 3-5 天，跨境可接受 7-14 天但需追踪 | 无追踪信息、物流信息不更新 |
| 支付 | COD 占比高（ID/PH 超过 50%）、电子钱包 | 不支持 COD 直接损失大量订单 |
| 促销 | 9.9/10.10/11.11/12.12 是核心流量池 | 不参加平台大促 = 隐形 |
| 内容 | 直播带货、短视频种草 | 只做静态 listing，缺乏内容运营 |

### TikTok Shop（全球）

| 维度 | 买家期望 | 中国卖家常见差距 |
|------|---------|-----------------|
| 内容 | 有趣/有用/有共鸣的视频内容驱动购买 | 硬广思维，内容无趣 |
| 信任 | 达人/创作者推荐 > 品牌自说自话 | 不做达人合作，只投信息流广告 |
| 价格 | 冲动消费为主，客单价低于 Amazon | 高客单品在 TikTok Shop 转化差 |
| 体验 | 下单流畅、物流快、开箱体验好 | 包装廉价、物流慢、没有 unboxing 惊喜 |
| 风格 | 真实 > 精致、UGC 风格 > 广告片 | 用品牌宣传片的方式做 TikTok 内容 |

### 独立站（DTC / Shopify）

| 维度 | 买家期望 | 中国卖家常见差距 |
|------|---------|-----------------|
| 品牌故事 | 为什么做这个品牌、价值主张是什么 | 无 About 页面、品牌故事缺失 |
| 信任信号 | SSL、reviews、trust badge、退货政策清晰 | 网站像钓鱼站——无 reviews、无信任标 |
| 支付 | 本地支付方式（Klarna/Afterpay/PayPal 等） | 只支持信用卡，流失大量客户 |
| 网站体验 | 加载快、移动端友好、结账流程简洁 | 加载慢、移动端体验差、结账步骤多 |
| 内容 | Blog、使用指南、社交媒体整合 | 只有产品页，无内容生态 |
| 隐私 | Cookie 同意、隐私政策（EU 强制 GDPR） | 无 cookie banner、无隐私政策 = 违法 |

## 输出模式

### 快速文化诊断

用户说"这个产品在 XX 市场有没有问题"时，输出：
- 目标市场核心消费者特征（3 句话）
- 1-3 个本地化风险点
- 改进方向
- 控制在半页以内

### 完整市场洞察报告

用户要"系统做一次 XX 市场洞察"时，输出全部 5 个模块。

### 定向洞察

用户只需要某个维度（"日本消费者怎么买这类产品" / "Amazon listing 要注意什么" / "中东有什么禁忌"），只输出对应模块。

## 与其他技能的联动

| 联动技能 | 联动方式 |
|---------|---------|
| mc-listing | 市场洞察定义本地化方向 → listing 优化在此基础上执行 |
| mc-compliance | 文化禁忌 + 合规要求配合使用，避免上架后被下架 |
| mc-compete | 市场洞察提供消费者视角 → 竞品分析提供对手视角 |
| mc-ads | 市场洞察定义人群画像 → 广告投放精准定向 |
| mc-monitor | 市场洞察是基线 → 日常监控验证洞察是否仍然成立 |

## 交互规则

- 本技能可独立使用，也可作为其他技能的前置步骤
- 用户可以说"帮我做一次美国市场 × 瑜伽裤的消费者洞察"，系统输出完整报告
- 用户可以输入具体 listing/产品页，系统诊断本地化适配度
- 洞察结论可直接喂给 mc-listing、mc-ads、mc-compliance 使用
- 输出完成后告诉用户文件路径，不要在对话中重复全文
