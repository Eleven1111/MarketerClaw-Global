## Module 2: FBA 费用精算

### 2.1 Size Tier 判定

Amazon FBA 按产品尺寸和重量划分费用等级（2025 标准）：

| Size Tier | 最长边 | 中间边 | 最短边 | 重量 |
|-----------|-------|-------|-------|------|
| Small Standard | ≤ 15" | ≤ 12" | ≤ 0.75" | ≤ 1 lb |
| Large Standard | ≤ 18" | ≤ 14" | ≤ 8" | ≤ 20 lb |
| Large Bulky | ≤ 59" | ≤ 33" | ≤ 33" | ≤ 50 lb |
| Extra-Large (0-50 lb) | 超出 Large Bulky | — | — | ≤ 50 lb |
| Extra-Large (50-70 lb) | — | — | — | 50-70 lb |
| Extra-Large (70-150 lb) | — | — | — | 70-150 lb |
| Extra-Large (150+ lb) | — | — | — | > 150 lb |

> 尺寸取外包装尺寸（含包装材料），重量取 max(实际重量, 尺寸重量)。

尺寸重量计算：

```
尺寸重量(lb) = (长 × 宽 × 高)(英寸) / 139
计费重量 = max(实际重量, 尺寸重量)
```

> 仅 Large Standard 3lb+ 和所有 Oversize 使用尺寸重量规则。Small Standard 和 Large Standard < 3lb 使用实际重量。

### 2.3 仓储费

#### 月度仓储费

| 时期 | Standard Size | Oversize |
|------|--------------|----------|
| 1-9 月（非旺季） | $0.78/立方英尺 | $0.56/立方英尺 |
| 10-12 月（旺季） | $2.40/立方英尺 | $1.40/立方英尺 |

仓储费计算：

```
月仓储费 = 产品体积(立方英尺) × 月费率 × 月均库存量
产品体积(立方英尺) = 长(in) × 宽(in) × 高(in) / 1728
```

#### 长期仓储费（Aged Inventory Surcharge）

| 库龄 | 附加费 |
|------|--------|
| 271-365 天 | $0.50/立方英尺 或 $0.10/件（取大者） |
| 365+ 天 | $6.90/立方英尺 或 $0.15/件（取大者） |

### 2.4 低库存费（Low-Inventory-Level Fee）

2024 年新增。当 FBA 库存低于 28 天销量时触发：

| Historical Days of Supply | 附加费/件（Standard） |
|--------------------------|---------------------|
| 0-7 天 | $0.89 |
| 7-14 天 | $0.63 |
| 14-21 天 | $0.32 |
| 21-28 天 | $0.15 |
| 28+ 天 | $0（免费） |

> 仅适用于 Standard Size。新 ASIN（入仓 < 180 天）豁免。

### 2.5 入库配置服务费（Inbound Placement Service Fee）

Amazon 可能将库存分配到多个仓库。卖家可选择：

| 方案 | 费用 |
|------|------|
| Amazon 优化分仓（默认） | 免费（但库存分散到多仓） |
| 最少分仓（Minimal Shipment Splits） | 较低附加费/件 |
| 部分分仓（Partial Shipment Splits） | 中等附加费/件 |
| 单仓发货（Send to One Location） | 无此选项，必须至少接受部分分仓 |

费用按 Size Tier 和目的地数量浮动，Standard 件约 $0.21-0.68/件。

### 2.6 退货处理费（Returns Processing Fee）

2024 年 6 月起，对退货率超品类基准的品类收取：

| 品类 | 退货率基准 | 超出部分费用/件 |
|------|-----------|---------------|
| 服装 | ~25-30% | 与配送费相当 |
| 鞋类 | ~25% | 与配送费相当 |
| 电子产品 | ~10-15% | 与配送费相当 |
| 家居 | ~8-12% | 与配送费相当 |

> 仅对超出品类基准部分收费。基准内退货免费。
