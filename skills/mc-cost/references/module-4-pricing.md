## Module 4: 定价策略模型

### 4.1 成本加成定价法（Cost-Plus）

确定地板价（Floor Price）——低于此价必亏：

```
地板价 = 总 Landed Cost + 平台费用 + FBA 费用 + 最低广告费
       = 总成本 / (1 - Referral Fee% - 广告占比%)
```

更精确的公式：

```
目标售价 = 总成本 / (1 - 佣金率 - 目标利润率 - 广告占比)
```

示例：
```
总成本（采购+物流+关税+FBA+包装）= $8.50/件
佣金率 = 15%
目标利润率 = 20%
广告占比 = 10%（占售价）

目标售价 = $8.50 / (1 - 0.15 - 0.20 - 0.10)
         = $8.50 / 0.55
         = $15.45
→ 建议定价 $15.99（取整到 .99 或 .95）
```

### 4.2 竞争定价法（Competition-Based）

定位矩阵：

```
            高价
             │
    Premium  │  Value+
    (差异化)  │  (高性价比)
             │
 ────────────┼──────────── 品质/功能
             │
    Niche    │  Price War
    (小众精品) │  (价格战，慎入)
             │
            低价
```

操作步骤：
1. 搜集 Top 10 竞品价格（BSR 排名前 10 在该关键词下的产品）
2. 计算价格带：最低价、中位数、最高价
3. 定位策略选择：
   - **低于中位数 10-15%**：激进获客，牺牲利润换排名
   - **中位数附近**：平衡利润与竞争力
   - **高于中位数 15-30%**：需有明确差异化支撑（品牌、功能、包装）

### 4.3 价值定价法（Value-Based）

不看成本看价值——用户愿意为哪些差异付更多钱：

| 溢价因素 | 可支撑溢价 |
|---------|-----------|
| 品牌认知 | +10-50% |
| 独特设计/专利 | +15-40% |
| 更好材质 | +10-25% |
| Bundle/套装 | +20-60%（但边际成本增幅小） |
| 认证背书（Organic、BPA-Free） | +5-15% |
| 完善售后/保修 | +5-10% |

### 4.4 促销与 Coupon 对利润的影响

```
实际到手收入 = 售价 × (1 - Coupon折扣%) - Coupon 本身费用($0.60/次)
Referral Fee 按折后价计算
```

示例：
```
售价 $24.99，设 20% Coupon
折后价 = $24.99 × 0.80 = $19.99
Referral Fee = $19.99 × 15% = $3.00
Coupon 费用 = $0.60
实际到手 = $19.99 - $3.00 - $0.60 = $16.39
（对比无 Coupon：$24.99 - $3.75 = $21.24）
单件利润减少 = $21.24 - $16.39 = $4.85
```

> Lightning Deal 和 Best Deal 也有类似折扣费用，且有额外 merchandising fee。

### 4.5 Multi-Pack / Bundle 定价数学

```
Bundle 售价 = 单件价格 × 数量 × 折扣系数（通常 0.75-0.90）
Bundle 成本 = 单件成本 × 数量 + 额外包装费
Bundle 利润率通常 > 单件利润率（因为 Referral Fee 一次性、FBA 费按整包计）
```

示例：
```
单件：售价 $12.99，成本 $5.50，Referral Fee $1.95，FBA $3.65
→ 利润 = $1.89（14.5%）

3件装：售价 $29.99，成本 $16.50+$1.00包装，Referral Fee $4.50，FBA $5.92
→ 利润 = $2.07（6.9% per unit, but $6.07 total = 20.2% margin）
```

> Bundle 的关键优势：单次 FBA 费和 Referral Fee 比 3 次单件低很多。

### 4.6 价格弹性估算框架

```
价格弹性 ≈ (销量变化% / 价格变化%)
```

| 品类特征 | 弹性系数估计 | 含义 |
|---------|------------|------|
| 刚需、少竞品 | -0.5 ~ -1.0 | 提价 10%，销量降 5-10% |
| 一般消费品 | -1.0 ~ -2.0 | 提价 10%，销量降 10-20% |
| 高竞争、同质化 | -2.0 ~ -4.0 | 提价 10%，销量降 20-40% |

最优利润价格估算：

```
利润最大化价格 ≈ 边际成本 × |弹性| / (|弹性| - 1)
```

> 这是理论值，实际需要 A/B 测试验证。建议每次调价幅度不超过 5-8%。
