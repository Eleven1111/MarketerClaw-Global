## Module 5: 利润模拟器

### 5.1 单位经济模型（Unit Economics）

```
Revenue（营收）              = 售价 × 销量
 - COGS（产品成本）           = FOB成本 + 包装
 - Inbound Shipping（头程）   = 海运/空运分摊
 - Import Duty（进口关税）     = 关税 + Section 301
 - Platform Fee（平台费）      = Referral Fee
 - FBA Fee（配送费）           = Fulfillment + Storage
 - Advertising（广告）         = 售价 × ACOS × 广告销售占比
 - Returns（退货损耗）         = 售价 × 退货率 × (1 - 可二次销售率)
 - Other（其他）              = 保险 + 检测分摊 + 工具订阅
 ─────────────────────────────
 = Net Profit（净利润）

 Net Margin（净利率）= Net Profit / Revenue × 100%
```

#### 广告成本计算

```
广告成本/件 = 售价 × TACOS
TACOS = 总广告花费 / 总营收（含自然单）
```

| 产品阶段 | 典型 TACOS | 说明 |
|---------|-----------|------|
| 新品推广期（1-3 月） | 20-40% | 大量广告投入换排名 |
| 成长期（3-6 月） | 12-20% | 自然单逐步增长 |
| 成熟期（6-12 月） | 8-12% | 自然单占主导 |
| 收获期（12 月+） | 5-8% | 品牌效应，低广告依赖 |

#### 退货损耗计算

```
退货损耗/件 = 退货率 × [售价 × (1 - 可二次销售率) + 退货运费分摊]
```

| 品类 | 典型退货率 | 可二次销售率 |
|------|-----------|------------|
| 电子产品 | 8-15% | 60-70% |
| 服装 | 20-35% | 40-60% |
| 家居厨具 | 5-10% | 70-85% |
| 美妆个护 | 3-8% | 30-50% |
| 玩具 | 5-10% | 60-80% |
| 鞋类 | 15-30% | 50-65% |

### 5.3 保本分析（Break-Even Analysis）

```
保本售价 = 总固定成本/件 / (1 - 变动成本率)

其中：
  固定成本/件 = (FOB + 包装 + 头程 + 关税)
  变动成本率 = Referral Fee% + TACOS% + 退货率 × 损耗比

保本销量 = 月固定开支 / (售价 - 单件变动成本)

月固定开支包括：
  - 产品责任险分摊
  - 工具订阅（Helium 10, Jungle Scout 等）
  - 仓储基础费用
  - 人工成本分摊
```

### 5.5 现金流时间线

典型跨境电商资金周转周期：

```
时间线（从下单到回款）：

Day 0    ─── 向供应商下单（支付 30% 定金）
Day 30   ─── 生产完成（支付 70% 尾款）
Day 35   ─── 出厂 + 内陆运输到港
Day 40   ─── 海运起运
Day 60   ─── 到达目的港 + 清关
Day 65   ─── 入库 FBA 仓
Day 70   ─── 开始销售
Day 80   ─── 首批订单出货
Day 94   ─── Amazon 首次打款（14 天结算周期）

💰 资金占用周期：约 90-100 天（从付定金到首次回款）
🔄 完全回本周期：取决于动销速度，通常 120-180 天
```

资金需求计算：

```
首批资金需求 = 采购成本 × 首批数量
             + 头程物流费
             + 进口关税
             + 首月广告预算
             + 平台保证金（如适用）
             + 产品认证/检测费

周转资金 = 日均销售额 × 资金占用天数
```

### 5.7 ROI 计算

```
年化 ROI = (年净利润 / 总投入资金) × 100%

总投入资金 = 首批采购 + 首批物流 + 关税 + 首批广告
           + 品牌注册 + 产品认证 + 摄影/A+ 内容

月度 ROI = 月净利润 / 月均占用资金 × 100%
年化 ROI = 月度 ROI × 12
```

ROI 基准：

| ROI 水平 | 评价 |
|---------|------|
| < 50% 年化 | 偏低，考虑优化或换品 |
| 50-100% | 合格，中等水平 |
| 100-200% | 良好，值得加大投入 |
| 200%+ | 优秀，尽快扩量 |
