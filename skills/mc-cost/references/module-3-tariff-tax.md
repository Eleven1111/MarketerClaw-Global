## Module 3: 关税与税务估算

### 3.1 HS Code 查询指引

HS code（Harmonized System Code）是全球统一的商品分类编码，前 6 位国际通用，后续位数各国自定义。

查询方法：
1. **美国 HTSUS 查询**：https://hts.usitc.gov/ — 搜索产品英文关键词
2. **中国海关 HS 编码**：http://www.customs.gov.cn — 按中文品名查
3. **工具辅助**：搜索 `[产品名] HS code` 或 `[产品名] harmonized tariff`
4. **报关行确认**：最终以报关行核定为准，自查仅做参考

常见品类 HS code 范例：

| 产品 | HS Code (US) | 基础税率 |
|------|-------------|---------|
| 塑料制家居用品 | 3924.10 | 3.4% |
| 不锈钢厨具 | 7323.93 | 2-5.3% |
| 电子配件 | 8504.40 | 0-2.5% |
| 纺织品/服装 | 6109-6114 | 10-32% |
| 硅胶制品 | 3926.90 | 0-5.3% |
| 木制品 | 4420-4421 | 0-5.1% |
| 陶瓷制品 | 6911-6912 | 0-9% |
| 玩具 | 9503 | 0% |
| LED 灯具 | 9405.40 | 3.9% |
| 手机壳 | 3926.90 | 5.3% |

### 3.2 美国关税计算

```
应缴关税 = 申报价值 (CIF/FOB) × (MFN 基础税率 + Section 301 附加税率 + 反倾销税率)
```

#### Section 301 关税（中国产品附加税）

| 清单 | 加征税率 | 涵盖范围 |
|------|---------|---------|
| List 1 ($34B) | 25% | 工业机械、电子元件 |
| List 2 ($16B) | 25% | 半导体、化学品 |
| List 3 ($200B) | 25% | 消费电子、家具、家居用品 |
| List 4A ($120B) | 7.5% | 部分消费品、服装 |

> 2025 年部分品类税率有调整，具体以 USTR 最新公告为准。关键品类（电池、太阳能、钢铝、半导体）税率可能更高。

#### 反倾销税/反补贴税（AD/CVD）

部分品类存在额外的反倾销或反补贴税，常见高风险品类：
- 钢铁制品：25-500%+（视来源国和品类）
- 铝制品：各种幅度
- 木制品/家具：从中国进口有单独税率

> 必须在 https://www.usitc.gov 查询具体 AD/CVD 税令。

#### 关税计算示例

```
产品：硅胶厨具 (HS 3924.10)
申报价值：$5,000 (FOB)
MFN 基础税率：3.4%
Section 301 (List 3)：25%
实际关税率：3.4% + 25% = 28.4%
应缴关税：$5,000 × 28.4% = $1,420
单件关税：$1,420 / 1,000件 = $1.42/件
```

### 3.3 欧盟关税与 VAT

```
进口关税 = CIF 价值 × EU 关税税率
进口 VAT = (CIF 价值 + 进口关税) × VAT 税率
```

| 国家 | VAT 标准税率 | 说明 |
|------|-------------|------|
| 德国 | 19% | 部分商品 7%（食品、书籍） |
| 法国 | 20% | 部分商品 5.5% |
| 意大利 | 22% | — |
| 西班牙 | 21% | — |
| 荷兰 | 21% | — |
| 英国 (UK) | 20% | 已脱欧，独立关税制度 |

> EU 关税通常 0-17%，视品类。服装/鞋类较高（8-17%），电子品较低（0-6%）。
> VAT 可通过进项税抵扣机制回收（注册 VAT 号后）。

#### EU De Minimis

- 关税免征额：€150 以下货值免关税（商业进口不适用）
- VAT 免征额：2021 年起取消（所有货值均需缴 VAT）

### 3.4 日本关税与消费税

```
进口关税 = CIF 价值 × 日本关税税率
消费税 = (CIF 价值 + 进口关税) × 10%
```

| 品类 | 日本关税率 |
|------|----------|
| 电子产品 | 0% |
| 塑料制品 | 0-6.4% |
| 纺织品/服装 | 5-13.4% |
| 食品 | 各种（需检查个别品类） |
| 皮革制品 | 8-30% |

> 日本对多数消费品关税较低，但消费税统一 10%。

### 3.5 各国 De Minimis 门槛

| 国家/地区 | 关税免征额 | 税免征额 |
|----------|-----------|---------|
| 美国 | $800 | $800（同步） |
| 加拿大 | CAD $20 (关税) | CAD $40 (GST) |
| 英国 | £135 | £0（全部征 VAT） |
| 欧盟 | €150 | €0（全部征 VAT） |
| 日本 | ¥10,000 | 同步 |
| 澳大利亚 | AUD $1,000 | AUD $0（>$75 征 GST 10%） |

### 3.6 转让定价提示

跨境电商常见的降低关税策略：
- 合理分拆采购价与服务费（设计费、模具费等不计入关税基数）
- 利用 First Sale Rule（如适用，以第一道交易价格报关）
- 注意：低报关价属违法行为，风险极高
