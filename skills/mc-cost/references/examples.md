## 应用示例

### 示例：硅胶厨房铲套装（5 件套），中国出发 → Amazon US

**基础信息：**
- FOB 价格：$3.80/套
- 重量：450g（含包装 580g / 1.28 lb）
- 包装后尺寸：30 × 10 × 8 cm（11.8" × 3.9" × 3.1"）
- HS Code：3924.10（塑料制餐具和厨房用具）
- 目标月销量：500 件
- 物流方式：海运散货

**Step 1 — 成本拆解：**

```
① 产品成本 (FOB)                $3.80
② 包装（彩盒+FNSKU+Poly Bag）    $0.55
③ 头程海运（散货分摊）             $0.85
   (含海运 $0.40 + 清关 $0.15 + 拖车 $0.30)
④ 进口关税
   MFN: 3.4% + Section 301 List 3: 25% = 28.4%
   $3.80 × 28.4% = $1.08                $1.08
⑤ FBA Fulfillment Fee
   Size Tier: Large Standard (11.8" × 3.9" × 3.1", 1.28 lb)
   1-1.5 lb 档位                         $5.01
⑥ FBA 月仓储费
   体积 = 11.8 × 3.9 × 3.1 / 1728 = 0.083 立方英尺
   非旺季：0.083 × $0.78 = $0.065       $0.07
⑦ Inbound Placement Fee               $0.27
⑧ 合计 Landed + FBA 成本              $11.63
```

**Step 2 — 定价计算：**

```
竞品价格带：$14.99 - $24.99，中位数 $18.99

成本加成法：
  目标售价 = $11.63 / (1 - 0.15 - 0.15 - 0.10)
           = $11.63 / 0.60
           = $19.38
  → 建议 $19.99

三档方案：
  🔴 激进：$16.99 → 毛利 $2.82 (16.6%) — 冲排名用
  🟡 平衡：$19.99 → 毛利 $5.32 (26.6%) — 推荐
  🟢 高端：$22.99 → 毛利 $7.82 (34.0%) — 需差异化
```

**Step 3 — 月度 P&L（取平衡方案 $19.99 × 500 件）：**

```
Revenue                         $19.99 × 500 = $9,995
 - COGS + 包装                   $4.35 × 500 = $2,175  (21.8%)
 - 头程物流                       $0.85 × 500 = $425    (4.3%)
 - 关税                          $1.08 × 500 = $540    (5.4%)
 - Referral Fee (15%)            $3.00 × 500 = $1,500  (15.0%)
 - FBA Fee                       $5.35 × 500 = $2,675  (26.8%)
 - 广告 (TACOS 12%)              $2.40 × 500 = $1,199  (12.0%)
 - 退货损耗 (7% × 40%不可售)      $0.56 × 500 = $280    (2.8%)
 - 其他                          $0.20 × 500 = $100    (1.0%)
 ─────────────────────────────────────────────
 Net Profit                      $2.20 × 500 = $1,101  (11.0%)
```

**Step 4 — 敏感性：**

```
最敏感变量：TACOS
  TACOS 从 12% → 17%（+5%）：净利率从 11.0% → 6.0%（下降 5pp）
  TACOS 从 12% → 22%（+10%）：净利率从 11.0% → 1.0%（接近保本）

第二敏感：关税
  如 Section 301 取消：净利率从 11.0% → 16.4%（+5.4pp）
  如 Section 301 升至 50%：净利率从 11.0% → 6.2%（-4.8pp）
```

**Step 5 — 现金流：**

```
首批 1000 件投入：
  采购：$3,800（定金 $1,140 + 尾款 $2,660）
  物流 + 关税：$1,930
  首月广告：$1,200
  其他：$500
  ─────────
  总投入：$7,430

回本时间线：
  月净利润 $1,101 → 约 6.7 个月回本
  年化 ROI = ($1,101 × 12) / $7,430 = 178%（良好）
```
