## Module 5: 供应链初判

### 5.1 供应商寻源

**寻源渠道优先级**：

| 渠道 | 适用场景 | 优势 | 劣势 |
|------|---------|------|------|
| 1688 | 成熟品类、现货 | 价格透明、起订量低、品种多 | 品质参差、需验厂 |
| Alibaba.com | 定制产品、外贸工厂 | 英文沟通、有信保 | 价格偏高、部分是贸易商 |

### 5.2 成本结构基准

**产品成本构成**：

```
产品出厂价 (FOB)
  = 原材料成本
  + 生产加工费
  + 包装费（内包装 + 外箱）
  + 品质检验费（验货费分摊）
  + 工厂利润（通常 10%-20%）

到手成本（Landed Cost）
  = FOB 价
  + 国内运输到港/仓
  + 报关/商检费
  + 海运/空运费
  + 目的国关税
  + 目的国清关费
  + 尾程派送到 FBA 仓
```

**成本占比基准（Amazon FBA 标准件）**：

| 成本项 | 占售价比例 | 备注 |
|-------|----------|------|
| 产品成本（FOB） | 15%-30% | 目标 <25% |
| 头程物流（到 FBA） | 5%-15% | 海运约 5%-8%，空运约 10%-15% |
| FBA 配送费 | 10%-20% | 按体积重量计算 |
| 平台佣金 | 15% | Amazon 大部分品类 |
| 广告费 | 8%-20% | 新品期偏高 |
| 其他费用 | 3%-5% | 仓储、退货、损耗等 |
| **目标净利润** | **15%-30%** | **低于 15% 不建议做** |

### 5.4 物流复杂度评估

| 因素 | 低复杂度（1 分） | 中复杂度（3 分） | 高复杂度（5 分） |
|------|-----------------|-----------------|-----------------|
| 尺寸 | 标准件（<18"任一边） | 大标准件 | 超大件/超重 |
| 重量 | <2 lbs | 2-20 lbs | >20 lbs |
| 易碎性 | 不易碎 | 一般易碎 | 极易碎（玻璃/陶瓷） |
| 带电池 | 无电池 | 内置锂电池 | 纯电池/超大电池 |
| 液体/粉末 | 无 | 含少量液体 | 大量液体/气雾剂 |
| 温度敏感 | 不敏感 | 需避免极端温度 | 需冷链 |
| 危险品分类 | 非危险品 | 有限数量危险品 | 完全危险品 |

**物流复杂度评分**：

```
物流复杂度分 = 各因素最高分

1-2 分 → 简单物流，FBA 标准流程
3 分 → 中等复杂，需要注意包装和申报
4 分 → 较复杂，需要专业货代，可能有额外费用
5 分 → 高复杂，需评估是否值得做（物流成本可能吞噬利润）
```

### 5.5 FBA vs FBM vs 3PL 决策

| 条件 | FBA（推荐默认） | FBM | 海外仓 3PL |
|------|----------------|-----|-----------|
| 产品体积 | 标准件 | 超大件 | 大件/重件 |
| 日单量 | >5 单/天 | <5 单/天（测品期） | 10+ 单/天 |
| 售价 | >$15 | <$10（FBA 费占比过高） | >$20 |
| 退货率 | <5% | >10%（FBM 可控制） | 5%-10% |
| 时效要求 | 需要 Prime 标 | 不需要 Prime | 需要快速配送 |
| 季节性 | 全年稳定 | 季节性强（避免长期仓储费） | 全年有量 |
| 定制化 | 标准品 | 需要定制/个性化 | 需要贴标/改包装 |

### 5.6 首批交货时间估算

| 阶段 | 标准品（现模） | 微改款 | 定制开模 |
|------|-------------|--------|---------|
| 找供应商+询价 | 3-5 天 | 5-7 天 | 7-14 天 |
| 谈判+下样品单 | 2-3 天 | 3-5 天 | 5-7 天 |
| 样品制作 | 3-7 天 | 7-15 天 | 30-60 天（含开模） |
| 样品确认+改版 | 3-7 天 | 7-14 天 | 14-30 天 |
| 大货生产 | 7-15 天 | 15-30 天 | 30-45 天 |
| 品检+发货 | 3-5 天 | 3-5 天 | 5-7 天 |
| 海运到港 | 15-30 天 | 15-30 天 | 15-30 天 |
| 清关+入 FBA | 5-10 天 | 5-10 天 | 5-10 天 |
| **总计** | **40-80 天** | **60-110 天** | **110-200 天** |

**空运可将物流环节压缩至 5-7 天，但成本增加约 3-5 倍。**
