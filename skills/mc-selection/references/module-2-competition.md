## Module 2: 竞争格局分析

### 2.1 品类集中度分析

**核心指标：CR10（Top 10 卖家市场份额）**

```
CR10 = Top 10 卖家月销售额之和 ÷ 品类总月销售额 × 100%

> 70% → 高度集中，寡头市场（进入难度极高）
40%-70% → 中等集中（有机会但需差异化）
< 40% → 分散市场（新卖家友好）
```

**品牌 vs 白牌比例**：

| 指标 | 新卖家友好 | 中性 | 不友好 |
|------|-----------|------|-------|
| Top 20 中品牌占比 | <30% | 30%-60% | >60% |
| 品牌注册卖家占比 | <40% | 40%-70% | >70% |
| 头部是否有亚马逊自营 | 否 | 有但不在 Top 5 | Top 3 有亚马逊自营 |

### 2.2 进入壁垒评估

逐项评估以下进入壁垒，每项 1-5 分（1=无壁垒，5=极高壁垒）：

**资金壁垒**：

| 分数 | 首批备货成本（含 FBA 头程） | 说明 |
|------|---------------------------|------|
| 1 | <$2,000 | 轻资产入场 |
| 2 | $2,000-$5,000 | 小额投入 |
| 3 | $5,000-$15,000 | 中等投入 |
| 4 | $15,000-$50,000 | 较大资金量 |
| 5 | >$50,000 | 重资产品类 |

**认证壁垒**：

| 分数 | 所需认证 | 说明 |
|------|---------|------|
| 1 | 无需认证 | 普通日用品 |
| 2 | 简单合规（CPC 等） | 儿童产品基础认证 |
| 3 | 标准认证（UL/FCC 等） | 电子产品常规认证 |
| 4 | 严格认证（FDA/EPA 等） | 食品接触、杀虫类 |
| 5 | 极高门槛（医疗器械、处方类） | 基本不建议新手 |

**知识产权壁垒**：

| 分数 | IP 风险程度 | 说明 |
|------|-----------|------|
| 1 | 品类通用，无 IP 风险 | 基础日用品 |
| 2 | 存在少量外观专利 | 需避开特定设计 |
| 3 | 品类有活跃的 IP 诉讼 | 需专利排查 |
| 4 | 核心功能受专利保护 | 创新空间受限 |
| 5 | 品类被少数专利覆盖 | 极高侵权风险 |

**MOQ 壁垒**：

| 分数 | 起订量 | 说明 |
|------|-------|------|
| 1 | <100 件 | 极低门槛 |
| 2 | 100-500 件 | 小批量测试可行 |
| 3 | 500-2,000 件 | 中等起订量 |
| 4 | 2,000-5,000 件 | 需较大资金 |
| 5 | >5,000 件 | 需工厂关系或大资金 |

**进入壁垒综合评分**：

```
进入壁垒分 = (资金壁垒 × 0.3) + (认证壁垒 × 0.25) + (IP 壁垒 × 0.25) + (MOQ 壁垒 × 0.2)

1.0-2.0 → 低壁垒（新手友好）
2.0-3.0 → 中等壁垒（需准备）
3.0-4.0 → 高壁垒（中阶以上卖家）
4.0-5.0 → 极高壁垒（资深卖家 + 资源匹配）
```

### 2.3 Review 壁垒分析

Review 数量是新卖家最直观的竞争门槛。

**Review 壁垒评估表**：

| 指标 | 低门槛 | 中门槛 | 高门槛 | 极高门槛 |
|------|-------|-------|-------|---------|
| Top 10 平均 Review 数 | <200 | 200-1,000 | 1,000-5,000 | >5,000 |
| 首页最低 Review 数 | <50 | 50-200 | 200-500 | >500 |
| Top 10 平均评分 | <4.0 | 4.0-4.3 | 4.3-4.5 | >4.5 |
| 新品上首页所需 Review | <30 | 30-100 | 100-300 | >300 |

**新卖家 Review 追赶速度估算**：

```
合规 Review 获取速度:
  Vine 计划: 约 30 个/批次（需 Brand Registry）
  自然 Review 率: 订单量的 1%-3%
  Request a Review 按钮: 可提升 Review 率至 3%-5%

追赶时间 = (首页最低 Review 数 - 30) ÷ (月销量 × Review 率)
```

**Review 壁垒评分**：

| 分数 | 追赶时间估算 | 建议 |
|------|------------|------|
| 1 | <2 个月 | 极低门槛，快速切入 |
| 2 | 2-4 个月 | 低门槛，正常运营可追上 |
| 3 | 4-8 个月 | 中等门槛，需要推广配合 |
| 4 | 8-18 个月 | 高门槛，需广告+Vine+站外组合 |
| 5 | >18 个月 | 极高门槛，不建议正面竞争 |

### 2.4 价格带分析

**价格带分布表**：

|---------|-----------|------|-------------|--------------|------|

| ${价格2} - ${价格3} | | | | | |
| ${价格3} - ${高价} | | | | | |

**价格带机会识别**：

```
利润空间丰厚的价格缺口 = 搜索量大 + 该价格带商品数量少 + 利润率 >30%

常见机会模式:
1. 中间断档: $15-20 很多，$30-40 很多，但 $20-30 空缺
2. 品质升级: 低价区(<$15)商品多但评价差，高品质中价位有机会
3. 套装策略: 单品红海，但套装/组合包价格带有空间
```

**利润模型快速估算**（Amazon FBA 2026）：

```
售价: ${X}
  - 产品成本: ${成本}（含包装，建议控制在售价 25%-30%）
  - 头程物流: ${头程}（含关税，2026 关税显著上升，需单独核算）
  - FBA 配送费: ${FBA fee}（按 Amazon 最新费率，2026 已多次上调）
  - 佣金(15%): ${佣金}
  - 广告费(目标 ACoS {X}%): ${广告}（PPC 竞争加剧，预留售价 15%-20%）
  - 退货损耗: ${退货}（按品类平均退货率估算，通常 1%-5%）
  ─────────────────
  净利润: ${净利} ({净利率}%)

⚠️ 2026 净利润门槛（扣完所有成本后）:
  >35% → 优秀，广告弹性大，抗风险能力强
  28%-35% → 良好，可持续运营的最低健康线
  20%-28% → 危险区，任何费用上涨或广告效率下降都可能亏损
  <20% → 不建议进入，Amazon 费用结构不给你留余地

注意：Amazon 2026 平均抽走卖家收入的 50% 以上（FBA费+广告+佣金合计）
      你的利润必须从剩下的 50% 中来。毛利率 <40% 几乎意味着净利润 <15%。
```
