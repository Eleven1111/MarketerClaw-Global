## Module 2：TikTok Shop 内容策略

### 2.1 内容类型与转化效率

| 内容类型 | 转化率 | 制作难度 | 建议占比 |
|---------|--------|---------|---------|
| **产品演示视频** | ⭐⭐⭐⭐⭐ | 低 | 40% |
| **问题解决型** | ⭐⭐⭐⭐ | 低 | 25% |
| **对比测试** | ⭐⭐⭐⭐ | 中 | 15% |
| **开箱 / 上手** | ⭐⭐⭐ | 低 | 10% |
| **生活方式植入** | ⭐⭐⭐ | 高 | 10% |

### 2.2 高转化视频结构（15–30 秒）

```
[0–3秒]  HOOK：提出问题或展示痛点
          例："你还在用这种 [竞品/旧方法] 吗？"
          例：直接展示"问题存在的样子"

[3–12秒] SOLUTION：展示产品如何解决问题
          核心功能 1–2 个，用动作展示而非语言描述

[12–22秒] PROOF：社交证明
          "已有 X 万人在用" / 真实用户反馈截图

[22–28秒] CTA：行动号召
          "点击购物车直接下单" / "链接在简介"
```

### 2.3 账号冷启动 SOP（前 30 天）

```
第 1 周：内容测试
□ 发布 5–7 条不同角度的产品视频
□ 格式：竖屏 9:16，720p 以上，字幕必加
□ 不要一开始就挂购物车（先让算法学习账号类型）
□ 每天固定时间发（美国 EST 12:00–15:00 或 19:00–22:00）

第 2 周：找到起量视频
□ 观察哪条视频完播率 > 40%、互动率 > 5%
□ 复制起量视频的结构，换角度再发 3–5 条
□ 开始挂购物车链接

第 3–4 周：放大
□ 给起量视频投 DOU+（预算 $50–$100 测试）
□ 开始联系达人合作（发起量视频给达人做参考）
```

## Module 3：达人 / KOL 合作框架

### 3.1 达人分层策略

| 类型 | 粉丝量 | 特点 | 合作模式 | 参考费用（美国市场） |
|------|--------|------|---------|-----------------|
| **纳米达人** | 1K–10K | 垂直粉丝，互动率高 | 免费寄样 / 纯佣金 | $0–$50/条 |
| **微型达人** | 10K–100K | 性价比最高 | 样品 + 小额固定费 | $50–$500/条 |
| **中型达人** | 100K–1M | 曝光量大，效果稳定 | 固定费 + 佣金 | $500–$5,000/条 |
| **头部达人** | 1M+ | 品牌背书价值高 | 纯固定费 | $5,000–$50,000+ |

**新卖家建议**：优先合作 **20–50 个纳米/微型达人**，比花同等预算找 1 个中型达人效果更可控。

### 3.2 达人合作 SOP

**寻找达人（免费方法）：**

```
方法 1：TikTok Creator Marketplace
路径：TikTok for Business → Creator Marketplace → 按品类/粉丝量/互动率筛选

方法 2：竞品评论区挖掘
→ 搜索竞品产品名 → 找发过相关内容的小达人
→ 查看主页 → 有商务邮箱则直接联系

方法 3：Amazon Influencer Program 联动
→ 已有 Amazon 影响力的创作者通常愿意接新品合作
```

**合作邀约模板（英文）：**

```
Subject: Collaboration Opportunity – [Product Name] for [Creator Name]

Hi [Name],

I love your content on [specific video/topic] — your audience is exactly 
who we're trying to reach.

We make [product in one sentence — benefit first, not features].
We'd love to send you a free sample with no strings attached. 
If you enjoy it, we offer [X% commission / $XX flat fee] for a video.

No scripts, no pressure — your honest take is what matters.

Interested? Reply and I'll send over the details.

Best,
[Your name] | [Brand]
```

### 3.3 佣金结构设计

```
推荐结构（平衡风险与吸引力）：

纳米达人：15–20% 佣金（无固定费）
微型达人：$50–$100 固定 + 10–15% 佣金
中型达人：$300–$1,000 固定 + 5–10% 佣金

追踪工具（免费）：
- TikTok Shop Affiliate 内置追踪
- Amazon Influencer 专属链接（卖家发邀请）
- UTM 参数 + Google Analytics（独立站）
```

## Module 4：UGC 体系（用户生成内容）

### 4.1 触发 UGC 的四个时机

```
时机 1：包裹开箱体验
  → 包装内放卡片："拍开箱视频 @我们，获得 $5 优惠券"
  → 不要要求好评（违反 ToS），只鼓励分享体验

时机 2：产品使用后 7–14 天
  → FeedbackWhiz / 或 Amazon Request a Review 发送后续邮件
  → 邀请分享社媒内容

时机 3：Facebook/Instagram 私信满意客户
  → 找到已自发分享内容的买家 → 授权 + 制作素材

时机 4：TikTok Duet / Stitch
  → 发布"邀请用户对我们的视频做 Duet"的内容
  → 成本为零，优质 UGC 自动产生
```

### 4.2 UGC 内容授权模板

```
授权消息模板（私信发送）：

Hi [Name], 

We saw your post about [product] and absolutely loved it! 
Would you be okay with us sharing it on our official channels 
with full credit to you?

We'd also love to offer you a [discount code / free product] 
as a thank-you.

Please reply "YES" if you're happy for us to use it.

[Brand Name]
```

## Module 5：Pinterest 导流策略（Amazon 专项）

Pinterest 是搜索型平台，用户带有明确购买意图，适合高客单价、视觉型产品。

### 5.1 Pinterest → Amazon 导流设置

```
Step 1：创建 Pinterest Business Account（免费）
Step 2：验证网站（建议用独立站做中转，提升 SEO 权重）
Step 3：创建产品图钉（Product Pin）
  → 图片：2:3 竖版（1000×1500px），白底或场景图
  → 标题：含核心关键词（Pinterest 有独立搜索引擎）
  → 描述：200字以内，含 2–3 个关键词
  → 链接：直接指向 Amazon Listing 或独立站产品页
Step 4：加入相关 Group Board（增加曝光）
Step 5：发布频率：每天 5–15 个图钉（可用 Tailwind 免费版排期）
```

### 5.2 Pinterest SEO 关键词策略

```
关键词来源（Pinterest 原生）：
1. Pinterest 搜索框自动补全
2. 搜索结果页顶部的"气泡标签"（Pinterest 推荐关键词）
3. 竞品 Board 的标题和描述用词

布局位置：图钉标题 > 图钉描述 > Board 名称 > Board 描述

示例（以收纳盒为例）：
图钉标题："Bamboo Kitchen Organizer – Under Sink Storage Solution"
描述："Maximize your cabinet space with this bamboo organizer. Perfect for 
under sink storage, bathroom shelves, and pantry organization. Adjustable 
dividers, easy assembly. Shop on Amazon."
```

## Module 6：达人筛选框架（新增）

### 6.1 四维评估模型

在 Module 3 的分层策略基础上，用四维模型精准筛选达人：

| 维度 | 权重 | 评估标准 | 数据来源 |
|------|------|---------|---------|
| **粉丝质量** | 30% | 互动率（点赞+评论 / 粉丝数）≥3%、粉丝地域匹配目标市场 | TikTok Creator Marketplace / 手动检查 |
| **内容匹配度** | 30% | 近 30 条内容中 ≥50% 与你的品类相关 | 手动浏览主页 |
| **报价合理性** | 20% | CPE（Cost Per Engagement）≤$0.10（纳米）/ ≤$0.05（微型） | 报价 / 平均互动量 |
| **商业信誉** | 20% | 按时交付、内容质量稳定、无争议历史 | 合作记录 / 品牌口碑 |

## Module 7：UGC 飞轮设计（新增）

在 Module 4 的 UGC 触发时机基础上，设计可持续运转的 UGC 飞轮：

```
UGC 飞轮（4 步闭环）
━━━━━━━━━━━━━━━━━━━━━
① 种子内容
   → 品牌发布 3-5 条示范内容
   → 设定内容模板（开箱/测评/before-after）
   
     ↓
     
② 用户参与
   → 包裹卡引导 + 社媒活动
   → Hashtag Challenge（TikTok）
   → "Tag us for a chance to be featured"
   
     ↓
     
③ 内容沉淀
   → 筛选优质 UGC（画质、真实性、情感共鸣）
   → 获取授权（DM 模板见 Module 4）
   → 建立 UGC 素材库
   
     ↓
     
④ 二次分发
   → 官方账号转发/Repost
   → 广告素材二次利用（Meta/TikTok 广告）
   → Amazon A+ Content 嵌入
   → 独立站 Social Proof 区域
   
     → 回到 ①，新一轮种子内容...
━━━━━━━━━━━━━━━━━━━━━
```

飞轮启动指标：
- 每月新增 UGC ≥10 条 = 飞轮运转中
- 每月新增 UGC <3 条 = 需要加大 ② 的激励

## Module 8：达人合作 SOP（新增）

从触达到效果追踪的完整 SOP：

```
达人合作全流程 SOP
━━━━━━━━━━━━━━━━━━━━━

Step 1：达人发现（每周 2-3 小时）
□ TikTok Creator Marketplace 筛选
□ 竞品评论区挖掘相关达人
□ Hashtag 搜索找活跃创作者
□ 用四维评估模型打分（Module 6）

Step 2：触达邀约（每批 20-30 个）
□ 用合作邀约模板发送（见 Module 3）
□ 跟进：48h 未回复 → 二次触达
□ 预期回复率：10-20%

Step 3：谈判要点
□ 明确交付物：视频数量、时长、平台
□ 明确时间线：拍摄 → 初审 → 修改 → 发布
□ 佣金结构（见 Module 3 佣金设计）
□ 内容审核权：品牌有 1 次修改机会
□ 授权范围：是否可用于广告投放

Step 4：内容简报（Brief）
□ 产品核心卖点（不超过 3 个）
□ 目标受众描述
□ 必须包含的元素（产品使用场景）
□ 禁止的内容（虚假宣传、竞品对比）
□ 参考视频链接（2-3 条）
□ 不写脚本——让达人用自己的风格

Step 5：效果追踪（发布后 7-14 天）
□ 播放量 / 互动量 / 互动率
□ 通过 TikTok Shop Affiliate 或 UTM 追踪转化
□ ROI 计算：（销售额 - 成本 - 达人费用）/ 达人费用
□ 效果好的达人：加入长期合作池
□ 效果差的达人：分析原因，调整 Brief 或更换
━━━━━━━━━━━━━━━━━━━━━
```

💰 预算参考：20 个纳米达人合作 ≈ $0-$1,000（含样品）  ⏱ 4-6 周见效  👤 卖家/VA
