## Module 2: FBA vs FBM vs 3PL 决策

### 2.1 FBA（Fulfillment by Amazon）

Amazon 官方履约服务，卖家将库存发送到 Amazon 仓库，由 Amazon 负责存储、拣货、包装、配送和客服。

#### FBA 优势

| 优势 | 详细说明 | 量化影响 |
|------|---------|---------|
| **Prime 标识** | 自动获得 Prime badge，覆盖 2 亿+ Prime 会员 | 转化率提升 20%-50% |
| **Buy Box 加权** | FBA 在 Buy Box 算法中有权重加成 | Buy Box 赢率提升 30%+ |
| **配送时效** | 美国境内 1-2 天达（Prime），客户体验极佳 | 退货率降低，好评率提升 |
| **客户信任** | "Sold by X, Fulfilled by Amazon" 增强信任 | 新品牌尤为关键 |
| **客服外包** | Amazon 处理退换货和客户问题 | 节省人工成本 |
| **多渠道履约（MCF）** | 可用 FBA 库存为独立站/其他平台发货 | $5-$8/单（附加费用） |
| **移除/清算** | 滞销品可申请移除或 Amazon 清算 | 止损手段 |

#### FBA 劣势与风险

| 劣势 | 详细说明 | 应对策略 |
|------|---------|---------|
| **仓储费高** | 月度仓储费 + 超龄附加费（365天+） | 严格备货计划，避免超龄 |
| **长期仓储费** | 超 181 天：$0.50/立方英尺；超 271 天：$1.50；超 365 天：$6.90 或 $0.15/件取大 | 定期清理滞销库存 |
| **库存混放风险** | 默认 Commingling（混放），可能收到假货投诉 | 申请 Sticker Label（贴标隔离） |
| **库容限制** | IPI < 400 会被限制补货量 | 维持 IPI > 500 |
| **退货宽松** | Amazon 退货政策宽松，退货率高于 FBM | 提升 Listing 准确性减少退货 |
| **入库费用** | 入库配置费 + 放置费（2024 年新增） | 提前了解费率变动 |
| **断货风险** | 补货链路长，一旦断货排名暴跌 | 安全库存 + 多渠道备份 |

#### FBA 费用结构（Amazon US 参考）

| 费用项 | 计费方式 | 参考金额 |
|--------|---------|---------|
| 配送费（Fulfillment Fee） | 按尺寸+重量分级 | $3.22-$10+ / 件（Standard） |
| 月度仓储费（Monthly Storage） | 按立方英尺/月 | $0.87（1-9月）/ $2.40（10-12月） |
| 超龄仓储费（Aged Inventory） | 超 181/271/365 天加收 | $0.50-$6.90/立方英尺 |
| 入库配置费（Inbound Placement） | 按分仓情况计费 | $0.21-$1.58/件 |
| 移除/清算费 | 按件计费 | $0.97-$1.78/件 |
| 退货处理费 | 特定品类加收 | $2.12-$5+/件 |
| 低库存费（Low Inventory Level Fee） | 历史天数 < 28 天 | $0.32-$0.47/件 |

### 2.2 FBM（Fulfilled by Merchant）/ SFP（Seller Fulfilled Prime）

卖家自行或通过第三方物流履约配送。

#### FBM 适用场景

| 场景 | 原因 | 示例品类 |
|------|------|---------|
| **大件/超重产品** | FBA 费用极高，FBM 更经济 | 家具、健身器材、大型家电 |
| **低动销 SKU** | 避免长期仓储费 | 长尾配件、季节性产品 |
| **定制化产品** | 需要按订单生产/组装 | 定制 T 恤、刻字饰品 |
| **高退货率品类** | FBM 可控制退货流程 | 服装（退货率 20-30%） |
| **利润极薄** | FBA 费用侵蚀利润 | 低价小件（售价 < $10） |
| **多变体大量 SKU** | FBA 入库管理复杂 | 手机壳（数百变体） |

#### SFP（Seller Fulfilled Prime）

- **定义**：卖家自行配送但获得 Prime 标识
- **要求**：2 天内配送、99%+ 准时率、<0.5% 取消率
- **优势**：Prime badge + 库存自主权
- **劣势**：门槛高，需要本地仓库或可靠 3PL
- **现状**：Amazon 周期性开放 SFP 注册，竞争激烈

### 2.3 3PL / 海外仓

第三方物流（3PL）提供海外仓储、代发和增值服务。

#### 3PL 优势

| 优势 | 说明 |
|------|------|
| **灵活性** | 不受 FBA 库容限制，可存大量库存 |
| **成本可控** | 仓储费通常低于 FBA，无超龄附加费 |
| **多平台履约** | 同一仓库发 Amazon + Shopify + eBay + Walmart |
| **增值服务** | 换标、质检、组合包装、退货处理 |
| **中转功能** | 作为 FBA 补货的中转站（海外仓 → FBA） |
| **定制化** | 支持个性化包装、礼品卡、定制发票 |

#### 各区域主要 3PL 服务商

**美国市场：**

| 服务商类型 | 代表 | 特点 | 适合 |
|-----------|------|------|------|
| 华人海外仓 | 递四方、万邦、谷仓、出口易 | 中文沟通、价格友好 | 中国卖家首选 |
| 美国本土 3PL | ShipBob、ShipMonk、Red Stag | 全美分仓、技术集成好 | 品牌独立站 |
| Amazon 合作仓 | FBA 合作仓（AWD） | Amazon 官方中转仓 | FBA 卖家大批量入库 |

**欧洲市场：**

| 服务商类型 | 代表 | 特点 |
|-----------|------|------|
| 泛欧仓 | Byrd、Huboo、递四方欧洲仓 | 覆盖 DE/FR/IT/ES/UK |
| 英国仓 | 万邦英国仓、出口易英国仓 | 脱欧后英国独立仓库需求增加 |
| 波兰仓 | 成本低，覆盖中东欧 | 适合非主流站点 |

**日本市场：**

| 服务商类型 | 代表 | 特点 |
|-----------|------|------|
| 华人海外仓 | 递四方日本仓、云途日本仓 | 中文服务 |
| 日本本土 3PL | 佐川急便仓储、雅玛多 | 日本服务标准极高 |

**东南亚市场：**

| 服务商类型 | 代表 | 特点 |
|-----------|------|------|
| Shopee 官方仓 | Shopee Fulfillment（SBS） | Shopee 平台内优先推荐 |
| TikTok 官方仓 | TikTok Fulfillment by TikTok | TikTok Shop 内优先曝光 |
| 独立 3PL | 递四方东南亚仓、Locad | 多平台履约 |

#### 3PL 费用结构

| 费用项 | 计费方式 | 参考金额（美国） |
|--------|---------|-----------------|
| 入库费 | 按件/按箱 | $0.20-$0.50/件 |
| 仓储费 | 按 CBM/月或按托盘/月 | $5-$15/CBM/月 |
| 拣货费 | 按单/按件 | $1.50-$3.00/单 |
| 包装费 | 按件 | $0.50-$2.00/件 |
| 出库快递费 | 按重量/尺寸 | 与 UPS/USPS 协议价挂钩 |
| 退货处理费 | 按件 | $2.00-$5.00/件 |
| 增值服务 | 换标/质检/组合 | $0.10-$1.00/件 |

### 2.4 Shopee / TikTok Shop 履约方式

#### Shopee 物流选项

| 方式 | 说明 | 适用 |
|------|------|------|
| **Shopee Logistics（SPX）** | Shopee 自建物流，强制使用 | 大部分东南亚站点 |
| **SBS（Shopee Bao Service）** | Shopee 官方仓（类似 FBA） | 大卖家、高动销品 |
| **自发货** | 部分站点/品类允许 | 大件、定制品 |
| **跨境直发** | 中国仓直发到买家 | 新卖家、测品阶段 |

#### TikTok Shop 物流选项

| 方式 | 说明 | 适用 |
|------|------|------|
| **Fulfillment by TikTok（FBT）** | TikTok 官方仓，获得优先推荐 | 美国/英国站热销品 |
| **卖家自发货** | 从海外仓或国内直发 | 灵活，但时效要求严格 |
| **平台推荐物流** | 如 TikTok Shipping | 部分站点强制使用 |

### 2.5 履约模式决策框架

**决策树：**

```
产品月销量？
├── > 500 件/月（高动销）
│   ├── FBA 费用占售价 < 25%？
│   │   ├── 是 → FBA（主力方案）
│   │   └── 否 → 3PL + SFP（如有资格）
│   └── 产品超大件？
│       ├── 是 → 3PL / FBM
│       └── 否 → FBA
├── 100-500 件/月（中等动销）
│   ├── 利润率 > 30%？
│   │   ├── 是 → FBA
│   │   └── 否 → 3PL（成本更可控）
│   └── 变体数 > 50？
│       ├── 是 → 3PL（管理更灵活）
│       └── 否 → FBA
├── < 100 件/月（低动销）
│   ├── 售价 > $30？
│   │   ├── 是 → FBA（少量入库）
│   │   └── 否 → FBM / 3PL
│   └── 定制化产品？
│       ├── 是 → FBM（按单生产）
│       └── 否 → 少量 FBA + 3PL 备份
└── 新品测试
    └── 首批 FBA（100-200件）+ 剩余存 3PL
        └── 验证动销后再大批量入 FBA
```

**混合策略建议：**

| 策略 | 说明 | 适用 |
|------|------|------|
| **FBA 主力 + 3PL 缓冲** | Top 20% SKU 放 FBA，长尾放 3PL | 多 SKU 卖家 |
| **FBA + FBM 互补** | FBA 正常出货，断货时 FBM 顶上 | 防断货策略 |
| **3PL 中转 + FBA** | 大批量到 3PL 仓，少量多频转 FBA | 控制 FBA 仓储费 |
| **平台仓 + 自发** | Shopee SBS 热销品 + 跨境直发长尾品 | 东南亚多站点 |
