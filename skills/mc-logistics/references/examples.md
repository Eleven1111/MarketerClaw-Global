## 应用示例

### 示例：新品首次发 FBA（蓝牙耳机，Amazon US）

**背景：**
- 产品：TWS 蓝牙耳机（含锂电池）
- 单件重量：0.15kg，包装尺寸：12×8×5cm
- 售价：$29.99，成本：$8，目标毛利率：30%
- 预估月销：500 件（首月保守估计）
- 发货地：深圳

**Step 1：产品物流画像**

```
材积重 = 12×8×5 ÷ 5000 = 0.096kg
计费重 = MAX(0.15, 0.096) = 0.15kg（重货）
泡比 = 0.096 / 0.15 = 0.64（非抛货，无需优化包装）
Amazon 尺寸分级：Small Standard（12×8×5cm 折合约 4.7×3.1×2 英寸，< 15×12×0.75 英寸需再确认厚度）
特殊品：含锂电池，需 MSDS + UN38.3
```

**Step 2：头程方案推荐**

```
首批发货量：1,000 件（2 个月库存）
总重：1,000 × 0.15 = 150kg
总体积：1,000 × 0.00048 CBM = 0.48 CBM

推荐方案：空运专线（空派）
理由：
├── 货量小（150kg），海运不经济（LCL 起步费高）
├── 新品需要快速上架测试市场
├── 含电池，需走带电专线
└── 预估费用：150kg × $7/kg = $1,050

备选：空运 70%（700 件） + 海运 30%（300 件，测试海运渠道）
```

**Step 3：履约模式决策**

```
推荐：FBA
理由：
├── 售价 $29.99，FBA 配送费约 $3.22（Small Standard）
├── FBA 费用占售价 10.7%（< 25% 阈值）
├── 新品牌需要 Prime 标识建立信任
├── 蓝牙耳机竞争激烈，Buy Box 权重关键
└── 首批 1,000 件，仓储费可控
```

**Step 4：备货计划**

```
日均销量预估：17 件/天（500件/月）
补货前置时间：15 天（空运专线）+ 7 天（FBA 入库）= 22 天
安全库存：17 × 10 = 170 件
补货点：17 × 22 + 170 = 544 件
每次补货：17 × 45 = 765 件（覆盖 45 天）

首批入库：1,000 件
预计第 49 天库存降至 544 件 → 触发第二次补货
```

**Step 5：成本核算**

```
单件物流成本汇总：
├── 头程（空派）：$1,050 ÷ 1,000 = $1.05/件
├── FBA 配送费：$3.22/件
├── FBA 月仓储费：约 $0.02/件/月（Small Standard）
├── 入库配置费：约 $0.27/件
└── 总物流成本：约 $4.56/件

利润计算：
├── 售价：$29.99
├── Amazon 佣金（15%）：$4.50
├── 物流成本：$4.56
├── 产品成本：$8.00
├── 广告费（预估 15%）：$4.50
├── 毛利：$8.43/件
└── 毛利率：28.1%（接近 30% 目标，可接受）
```

**Step 6：风险评估**

```
风险 1：含电池产品清关风险
  概率：中  影响：高
  缓解：提前准备 MSDS + UN38.3，选有带电产品经验的货代

风险 2：新品销量不达预期
  概率：中  影响：中
  缓解：首批 1,000 件控制风险，45 天卖不完考虑 Coupon 促销

风险 3：空运价格波动
  概率：中  影响：低
  缓解：稳定后切换海运，长期成本下降
```
