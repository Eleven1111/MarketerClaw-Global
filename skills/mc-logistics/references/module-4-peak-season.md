### 4.4 旺季备货量计算

**旺季备货公式：**

```
旺季备货量 = (旺季日均销量 × 旺季持续天数) + 安全库存 - 当前可用库存 - 在途库存

其中：
旺季日均销量 = 常规日均销量 × 旺季放大倍数
旺季持续天数 = 大促前预热（7天） + 大促期（3-7天） + 大促后余温（7天）
```

**示例计算：**

```
产品：便携式充电宝
常规日均销量：50 件
BFCM 放大倍数：8x（参加 Deal）
旺季持续：7（预热） + 5（大促） + 7（余温） = 19 天

旺季日均销量 = 50 × 8 = 400 件（大促期）
  预热期：50 × 2 = 100 件/天 × 7 天 = 700 件
  大促期：50 × 8 = 400 件/天 × 5 天 = 2,000 件
  余温期：50 × 3 = 150 件/天 × 7 天 = 1,050 件
旺季总需求 = 700 + 2,000 + 1,050 = 3,750 件
安全库存 = 400 × 14 天 = 5,600 件（旺季安全库存加倍）

假设当前 FBA 库存：1,000 件，在途：500 件
备货量 = 3,750 + 5,600 - 1,000 - 500 = 7,850 件

分批发货：
  第一批（海运，8月底）：5,500 件（70%）
  第二批（海运，10月中）：1,600 件（20%）
  预留空运应急：750 件（10%）
```

### 4.5 超备 vs 欠备风险框架

| 维度 | 超备风险 | 欠备风险 |
|------|---------|---------|
| **资金** | 大量资金积压在库存中 | 错失旺季销售，少赚 |
| **仓储费** | 旺季仓储费翻倍（$2.40/立方英尺） | 无 |
| **超龄风险** | 旺季后卖不完变滞销 | 无 |
| **排名影响** | 无直接影响 | 断货导致排名暴跌，恢复需数周 |
| **广告浪费** | 无 | 广告投放无法消化，预算浪费 |
| **客户体验** | 无 | 缺货差评，影响品牌 |

**决策原则：**
- 热销品（Top 20% SKU）：宁可超备 20%，不冒断货风险
- 长尾品（Bottom 50% SKU）：宁可欠备，用 FBM 兜底
- 新品（无历史数据）：保守备货，空运应急

### 4.6 应急补货方案

当旺季发现备货不足时的应急措施：

| 紧急度 | 方案 | 时效 | 成本 | 适用 |
|--------|------|------|------|------|
| ★★★★★ | 国际快递直发 FBA | 5-7 天 | $8-$15/kg | 利润率高的热销品 |
| ★★★★ | 空运专线 + 快递派送 | 8-12 天 | $5-$9/kg | 中高利润品 |
| ★★★ | 3PL 海外仓转 FBA | 2-5 天 | $1-$3/件 | 有海外仓库存的 |
| ★★ | 切换 FBM 发货 | 即时 | 按单计费 | FBA 断货的临时替代 |
| ★ | 提前到货（催促货代） | 视情况 | 可能有加急费 | 在途货物 |
