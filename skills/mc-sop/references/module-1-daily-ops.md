## Module 1: 日常运营 SOP

> 跨境运营的核心不是做得多，而是做得准。每天的固定动作决定了 80% 的运营结果。

### 1.1 Amazon 卖家日常 SOP（亚洲时区）

按时间块组织，适用于中国/东南亚时区卖家运营美国站：

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
上午 9:00-11:00 · 数据检查 & 订单处理
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

□ 账户健康检查
  · Account Health Dashboard → 有无新警告/违规
  · 绩效通知 → 有无政策警告、ASIN 下架通知
  · Buyer Messages → 24h 内必须回复（影响 ODR）

□ 订单处理（FBM 卖家）
  · 新订单确认 → 打单发货
  · 异常订单排查（地址不完整、高风险订单）

□ 销售数据快览
  · 昨日销售额 / 订单量 / Session / CVR
  · 与上周同日对比，异常波动 > 20% 需排查
  · BSR 变动（核心 ASIN 排名是否稳定）

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
上午 11:00-12:00 · 广告 & 竞品
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

□ 广告数据检查
  · 昨日广告花费 / 销售 / ACOS
  · 异常 Campaign 识别（花费激增但无转化）
  · 预算消耗进度（是否提前花完或花不出去）

□ 竞品快速扫描
  · 核心竞品价格变动（是否降价/加 Coupon）
  · 竞品 Review 动态（新增差评可利用的机会）
  · 新进入者（搜索核心关键词看是否有新面孔）

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
下午 14:00-17:00 · Listing 优化 & 运营动作
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

□ Listing 优化（每天专注 1-2 个 ASIN）
  · 关键词排名追踪 → 排名下降的词需要加强广告
  · Search Term Report 分析 → 发现新词、加否定词
  · 主图/A+ Content 优化排期

□ Review 管理
  · 新 Review 查看（负评需要 24h 内回复）
  · Request a Review 操作（订单完成 5-30 天的订单）
  · Vine 计划管理（如有）

□ 库存监控
  · FBA 在途 / 在库 / 可售数量
  · 天数估算：当前库存 ÷ 日均销量 = 可售天数
  · < 14 天预警 → 触发补货流程

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
晚间 20:00-22:00 · 美国白天时段操作（可选）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

□ 广告竞价调整
  · 美国白天是高转化时段，可临时提高竞价
  · 检查 Top-of-Search 展示份额

□ Deal/Coupon 效果监控
  · Lightning Deal / Best Deal 进行中的实时表现
  · Coupon 核销率

□ 客户消息二次检查
  · 确保所有消息已在 24h 内回复
```

### 1.2 Shopee 卖家日常 SOP

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
Shopee 日常操作（与 Amazon 差异项）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

□ 聊天回复（Shopee Chat）
  · 回复率影响店铺评分，4 分钟内回复目标
  · 设置自动回复模板（常见问题）
  · 买家讨价还价应对策略

□ Shopee Feed 运营
  · 每日发布 1-2 条 Feed（产品使用场景、上新通知）
  · 关注竞品 Feed 动态

□ 促销工具管理
  · Flash Sale 报名（需提前 3-7 天）
  · Voucher 设置与更新
  · Bundle Deal 组合优化
  · Coins Cashback 活动监控

□ 物流与退货
  · 订单发货（Shopee 要求 2 天内发货，SLS 跟踪）
  · 退货/退款处理（Shopee 买家发起退货需 3 天内响应）
```

### 1.3 TikTok Shop 卖家日常 SOP

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
TikTok Shop 日常操作
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

□ 内容运营
  · 查看短视频数据（播放量、互动率、GMV 归因）
  · 达人合作进度跟踪（样品寄出、视频发布、佣金结算）
  · 新素材排期（每周至少 3-5 条产品视频）

□ 直播管理（如有）
  · 直播排期确认
  · 直播间话术 & 产品排序
  · 直播后数据复盘（GMV、观看人数、转化率）

□ Shop Tab 优化
  · 产品排序调整
  · 商品卡片标题/主图优化
  · 价格竞争力检查

□ 广告管理
  · Video Shopping Ads 数据检查
  · LIVE Shopping Ads 投放调整
  · Spark Ads（达人授权视频推广）效果追踪
```

### 1.4 周度运营节奏

| 周几 | 重点任务 |
|------|---------|
| 周一 | 上周数据复盘、本周目标设定、广告策略调整 |
| 周二 | Listing 优化日（1-2 个 ASIN 深度优化） |
| 周三 | 竞品分析日（mc-monitor / mc-compete） |
| 周四 | 库存 & 物流检查日（补货决策、在途跟踪） |
| 周五 | 新品准备日（选品研究、供应商沟通、样品评审） |
| 周六 | 学习 & 测试日（新功能测试、行业趋势研究） |
| 周日 | 休息 / 灵感收集 |
