# MarketerClaw Global

AI 驱动的跨境电商营销智能体。22 个技能覆盖从诊断到执行的全链路。

## 快速开始

在 Claude Code 中打开本项目，直接说：

- "帮我优化这个 Amazon Listing"
- "销量下降了，帮我诊断一下"
- "给我做个月度经营数据看板"
- "新品上线，帮我做 Launch 计划"
- "转化率太低了，查查哪里漏了"
- "蓝牙耳机在 Amazon US 上谁卖得最好"
- "分析一下 Anker 和 Baseus 的竞争策略"
- "我要投 Amazon PPC，预算 $5000/月，怎么分配"
- "怎么做 TikTok 内容，找达人合作"
- "帮我设计 Amazon 主图和 A+ 图片"
- "复购率太低了，怎么提升 LTV"
- "这个产品的 HS Code 是什么，关税多少"
- "3 个人的团队怎么安排日常运营"

## 系统特性

- **Brand Brain 数据总线**：所有技能共享品牌数据，消除信息孤岛
- **预设工作流**：6 套跨境场景自动串联技能（新品上线、增长突破、大促备战等）
- **危机模式**：现金流/Listing/库存三种危机自动检测和响应
- **季节感知**：自动识别跨境电商季节阶段，调整策略倾向
- **三级降级**：数据不足时自动降级，保守输出，不编造数据
- **成本标签**：每条建议附带预算、见效时间、执行人
